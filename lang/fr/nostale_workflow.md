# NosTale Workflow : Politiques et canaux Gstar

*Dernière mise à jour : 16/04/2026*

Ce fichier documente les workflows externes au jeu : distribution de contenu, canaux de communication, politiques d'échange avec les joueurs. À consulter quand un joueur pose une question qui touche aux services Gstar (nostar.fr, Insta, etc.).

---

## nostar.fr : codes d'accès aux guides

`nostar.fr/progress` héberge les guides de progression NosTale par acte (acte 1 à 9). L'acte 1 est en libre accès. Les actes 2 à 9 sont verrouillés derrière un code d'accès.

### Canal de distribution
Les codes sont envoyés uniquement par DM Instagram. Le compte Insta s'appelle `Codes NosTale` (Abd production). Les joueurs doivent suivre le compte puis envoyer un message privé.

### Quota
Chaque abonné Instagram reçoit 3 codes d'accès par semaine. Chaque code débloque le contenu exclusif (astuces OR/XP) d'un acte au choix.

### Politique d'échange Gstar
Avant d'envoyer les codes, Gstar demande au joueur de liker ses 6 dernières publications Instagram. Formulation type : "histoire que tu ne me sois pas redevable". L'idée c'est de garder l'échange mutuel plutôt qu'unilatéral.

### Comment répondre dans Discord
Quand un joueur demande un code ou de l'aide spécifique sur un acte (notamment 6+) :
- Ne pas tenter de répondre à la place du guide nostar (pas d'accès au contenu détaillé)
- Rediriger vers Insta DM
- Laisser Gstar décider lui-même de l'envoi du code (timing, échange de likes, etc.)
