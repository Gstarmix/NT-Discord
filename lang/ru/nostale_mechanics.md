## Прокачка по уровням

### Общий маршрут левелинга
- **lv65 → 85** : LoD (классический сервер), побочные квесты A5.1, основной квест A4
- **85 → H1** : начало героического уровня. Переход в акт 6. Героический уровень начинается после 85 уровня
- **H1 → H10** : A6 Адские руины 2 (путь без лучников Rement, точка возврата у входа), либо Воля Анкеллоан 3
- **H10 → H30** : A7 Загрязненный путь (осторожно с дебаффами мобов)
- **H30 → H60** : A8 первый LoL + Загрязненный путь для бонусного XP, A8 Bees начиная с H53
- **H60 → H67** : A8 второй LoL + карта A8 bees
- **H68 → H72** : A8 второй LoL верхняя карта + A8 sharks (смена карт штрафуется : 2 мин без баффа XP)
- **H73 → H80** : A8 второй LoL верхняя карта + Ледяной донжон (drakes до H83)
- **H80 → H90** : A9 третий LoL + карты (LoL требует хорошего сетапа, карты более щадящие, но смена карт штрафуется)
- **H90 → H99** : карты A10 (квест A10 дает H90 → H91)

### XP акт 9 (H80 → H90) : карты, рейт, ежедневки

**Рейтинг карт xp A9 :**
- Топ 1-2 (классические карты) : Загрязненная зона, Центральная
- Топ 3 (LoL) : Нижний уровень лаборатории, далее остальная часть LoL лаборатории

**Рейт XP H80 → H90 :**
- Дуо с хорошим сетом xp (божественная фея, медаль дворянина, аксессуары xp) в двойной карте : 11-13%/ч
- Двойной xp : рейт удвоен
- Соло без двойной карты : немного меньше
- 1-2ч карты xp в день = стабильный прогресс

**Ежедневки A9 :**
- 4 ежедневки/день = 3 в 9.1 + огромный абсолютный Arma в 9.2 (единственная ежедневка 9.2)
- 3 ежедневки 9.1 дают ~7%/день
- Основные квесты 9.1 с одного захода = ап примерно до 81
- Полный список : https://nosapki.com/fr/quests?category=153

**Выбор LoL против классических карт в A9 :**
- LoL A9 = хороший xp, слабый дроп
- Классические карты A9 (загрязненная зона, центральная, южный подземный путь) = интересный дроп для крафта стаффа 90, xp немного ниже

## Классические серверы (Dragonveil / Undercity)

- Постоянные серверы, именно там персонаж живет на длинной дистанции (после переноса с NF или при классическом старте)
- При создании персонажа выбор между стартом с 1 уровня или с 56 уровня со стаффом 50 уровня R5+7 в подарок
- Нет буста xp/золота (в отличие от NosFire)
- CI доступны с lv80

## Фарм LoD : сетап с несколькими мулами-магами

### Контекст в зависимости от сервера
- **Классический сервер** : старт lv55, квесты для подъема до 65, затем LoD с lv65 до lv80. Цель : достичь lv80, чтобы разблокировать CI. Оценка : 65→80 ≈ 4ч30 на мула
- **NosFire** : старт lv80 с SP2M в +5 в подарок в стартовом наборе. LoD служит только для подъема с lv80 до lv85. CI доступны только начиная с 85 уровня на NF (мера против ботов)

### Требования
- Семья обязательна для доступа к LoD (200k за создание)
- Помощник Боевой белый медведь (Bori) на основном SP1E, прокачанный до 60 уровня (59 единиц лучшего корма для питомцев), с куклами и премиум кормом для ресс
- Маунт с минимум 21 скорости передвижения (Кроличья карета идеальна ради защитного баффа, иначе Магический ягуар / Черепахолет / Rodéotoro за ~2M)
- Спасительные зелья и Вода из источника Циллоан
- Танковый стафф : Honorable : chaussures du maître voleur + Перчатка Магмароса в сумме 5 на NosBazar (120% сопротивления огню и тьме)

### Оптимальный сетап : 8 персонажей (классический сервер)
- Основной SP1E = танк, агрит сирен (Renés) своим маунтом
- Bori на основном : активирует Ярость берсерка во время агра (неуязвимость 1 мин, кулдаун 5 мин)
- 8-й персонаж (любой класс, мин +10) : его единственная роль = танковать Корну (Corne) своим петом. Заходит на карту последним → Корна целит последнего вошедшего
- 6 мулов-магов в группе : бьют + Групповое исцеление. Без помощника на мулах (мешает расстановке)

### Статы мулов
- **Сетап 6 мулов-магов** : никаких очков статов в SP2M (ни HP/MP, ни элемент). Причина : если Корна попнет на мулах, она ваншотит их всех разом → одновременная ресс + Корна меняет цель. Плавнее, чем поднимать по одному. Количества магов достаточно, чтобы зачистить агр без статов
- **Сетап команда 3 (1 escri + 1 маг SP2M + 1 лучник SP4A)** : вложить очки в элемент мага и лучника, чтобы быстрее чистить агры. Без статов слишком медленно с одним магом
- Избегать наступательных баффов : если мул превышает 10% урона по сирене, остальные мулы, бьющие менее 10%, теряют XP
- Танк SP1E : full HP/MP, без феи, при необходимости снять оружие. Цель = провоцировать, не нанося урона
- Видео-референс : https://www.youtube.com/watch?v=ERIzWLwQk5I

### Техника агра
- Правило : никогда не останавливаться (синхронизация NosTale не оптимизирована → удары на расстоянии при неподвижности)
- Маршрут : вход в драконий череп → отклониться вправо → идти вдоль правой стороны → срезать влево в конце → полукруг вправо → подобрать сирен слева → возврат к группе с движением туда-сюда вправо-влево

### XP и темп
- Бафф : Благословение богини Анселлоан + Карта Таро (Звезда) на каждом муле
- Костюм скелета : +30% к общему XP (10M на мула, 60M на 6)
- Корна попается примерно каждые 4 мин, Ярость берсерка кулдаун 5 мин
- При каждом повышении уровня профессии SP кулдаун навыков сбрасывается
- Оценка классический сервер : 65→68 за ~30 мин (с благословением + таро, паузами и Корной включительно). 65→80 ≈ 4ч30 на мула
- Золото : украшения LoD по 7k у PNJ → ~330k за сессию (пустой инвентарь)

### Почему маг для мультиаккаунта LoD
- Маг : SP2M доступна сразу, базовое AoE → проще всего
- Escrimeur : долгий квест SP3 на каждом муле
- Лучник : квест SP4 еще длиннее + у SP4A нет базового AoE

## Зарабатывание золота

- Фарм A3 для перьев и душ (до SP event)
- Рейды A4 + TS81 (нужна семья, TS81 менее выгоден, так как 0 золота за прохождение)

- Рейды : Ибрагим и Глацерус выгодны рано, A4/Лорена позже, A5/A7 (Кертос и Белиал), A8/A9 = много золота с дропами и материалами
- Квест SP Pet Trainer (нужно 500k репутации, гринд эссенций и петов, награды = репутация и золото)
- Дропы во время сессий XP (A5.1, A6, A7, A8, A9)
- Дропы вне сессий (те же карты + A5.2)
- NosMall (продажа ND-предметов, gamble box/wheel)
- Мини-игры в Миниленде ради материалов (Скумбрия очень востребована)

- **Классический сервер** : доступен начиная с 80 уровня
- Доступен на всех каналах
- Мгновенный бой против волн мобов

## Камни повышения SP

Три камня выдаются в конце основных квестов некоторых актов. Каждый поднимает SP сразу до 41+9 :
- **Gemme d'âme bleue / Blue Soulstone** (A4) → использовать на приоритетной SP класса
- **Gemme d'âme argentée / Silver Soulstone** (A6) → использовать на 2-й приоритетной SP
- **Gemme d'âme dorée / Golden Soulstone** (A7) → использовать на 3-й или 4-й SP

**Заметка NosFire** : синий камень души A4 не выдается на серверах NosFire (старт с 80 уровня, квест A4 уже пройден или награда убрана). Получить можно только серебряный (A6) и золотой (A7) камни.

## SP поддержки

Три SP могут играть роль поддержки, чтобы ассистировать основному игроку. Они получаются через квесты SP3 и SP4 (долгие).

### SP2M (Holy Mage) : поддержка хил + бафф
- Групповое исцеление (хил)
- Священное оружие (бафф dps)
- Благословение (бафф сопротивления)

### SP4A (Wild Keeper) : поддержка бафф + дебафф
Получается через квест SP4.
- Потеря Элементов (дебафф dps)
- Волк + Сияние стихий (бафф dps)
- Медведь (бафф сопротивления)

### SP3E (Crusader) : опциональная поддержка
Получается через квест SP3. Слабее, чем SP2M и SP4A.
- Молитва нападения (бафф dps)
- Защитная молитва (бафф сопротивления)
- Освящение (дебафф dps)

### Какая поддержка под твой класс
- **Escrimeur (SP1E) / Artiste martial (SP1AM)** : SP2M + SP4A. Классы ближнего боя, пет-танк не нужен.
- **Маг** : SP4A обязательно + пет обязательно (танк ближнего боя). SP3E опционально.
- **Лучник** : SP2M обязательно + пет обязательно (танк ближнего боя). SP3E опционально.

## Перенос персонажа (с NosFire)

- 1 персонаж на аккаунт через предмет из Battle Pass (высокая награда)
- Дополнительные переносы возможны при достижении 13 уровня баттл-пасса на других аккаунтах
- Выбор сервера назначения (предметы различаются по серверам, изменяемо до закрытия)
- Сервер виден в разделе "Расходники"
- Что переносится : все, что на персонаже (окно P), склад 10 слотов из BP, костюмы, инвентарь SP, активные рыночные предметы, миниленд (петы и декор), прогресс (квесты, репутация, татуировки, навыки, благополучие)
- Что НЕ переносится : золото (включая банк), семья, склад семьи, текущие аукционы (не возвращаются), список друзей, брак, черный список, рюкзак партнера
- Не полученные предметы NosMall : можно получить на сервере назначения
- Если ник уже существует на сервере назначения : обязательная смена при первом входе
- Невозможно изменить имя на NosFire (техническое ограничение)

### ESCRIMEUR

#### Основные SP (элементный квартет)
| Элемент | Основная SP | Альтернативы | Вывод |
|---|---|---|---|
| Огонь | Stone Breaker (SP11E) | Warrior (SP1E), Gladiateur (SP5E), Battle Monk (SP6E) | SP11E > SP1E > SP5E > SP6E |
| Вода | Ninja (SP2E) | Death Reaper (SP7E) | SP2E > SP7E |
| Тьма | Berserker (SP4E) | Renegade (SP8E), Waterfall Berserker (SP9E) | SP4E > SP8E > SP9E |
| Свет | Crusader (SP3E) | Dragon Knight (SP10E) | SP3E > SP10E |

#### Рекомендованные камни (Escrimeur)
- Blue Soulstone (A4) → Warrior (SP1E) : так как SP1E лучший доступный вариант рано
- Silver Soulstone (A6) → Berserker (SP4E)
- Golden Soulstone (A7) → Ninja (SP2E) или Stone Breaker (SP11E)

#### Сравнительные детали Escrimeur
- **SP1E (Warrior)** : огонь, бесплатна, доступна сразу. Провокация = агр мобов + базовое AoE = большое преимущество для пекса классических карт до H30. Затем заменяется на SP4E (LoL), SP3E (карты света) или SP11E (оптимальные огненные рейды). Сохраняет пользу в мультиаккаунте (агр) и поддержке-танке. Очки SP : микс атака/элемент до HL, full атака + остальное в элемент начиная с HL.
- **SP3E (Crusader)** : мощнее SP1E, доступна после квеста SP3. Приличное AoE, лучший вариант света на дистанции. Лучше для моббинга (мультицелевая автоатака полезна в групповом XP)
- **SP11E (Stone Breaker)** : огонь, очень мощная. Более затратное вложение, но лучшая огненная SP escrimeur
- **SP4E (Berserker)** : тьма, кэрри H30→H90. Очень хороший прогресс на дистанции
- **SP2E (Ninja)** : вода, хорошая альтернатива для водных карт

## Партнеры : когда начинать их использовать

- На низком уровне не играть с партнером. Использовать только помощника (Bori, усталый баран в зависимости от класса и контекста)
- Партнеры, дропающиеся в раннем гейме (Bob на TS25 и т.д.), посредственны и не успевают за прокачкой. Оставить для коллекции, но не для игры
- Интересоваться партнерами с момента приближения героического уровня (конец стандартного прогресса, переход в акт 6)
- Начиная с героического уровня : выбрать подходящего партнера (ближний или магический бой в зависимости от желаемой роли) и оптимизировать его через стафф партнера (см. следующий раздел)

---

## Стафф партнера : оптимизация танка

Цель стаффа партнера : максимизировать его выживаемость (сопротивления), а не урон.

### Оружие с сопротивлениями (по типу партнера)
- Партнеры ближнего боя (Ragnar, Kliff и т.д.) : оружие 65 и 71 уровня → +10 ко всем сопротивлениям
- Магические партнеры (Graham, Erdimine, Yertirand и т.д.) : оружие 63 уровня → +10 ко всем сопротивлениям

### Остальной стафф
- Броня : максимально высокого уровня, R0+5 минимум
- Аксессуары : мульти-сопротивления (например, морские)

### Контекст
- Вложение целесообразно только начиная с героического уровня
- На низком уровне партнер танкует без особого стаффа

## Помощники в раннем гейме (маг f2p)

- Роль помощника для мага : танковать мобов в ближнем бою, пока маг бьет на расстоянии
- Лучший бесплатный танк в раннем гейме (классический сервер) : усталый баран, ловится на TS23 начиная с 25 уровня
- В ожидании 25 уровня : использовать любого ловимого поблизости помощника
- На NosFire : Шоколадный кекс 16-летней годовщины (входит в Мощный пакет на старте за 330 NT) выгодно заменяет усталого барана или Bori в качестве первого помощника, уже застаффленный и доступный с самого начала
- Воскрешение помощника :
  - Семена власти : автоматическое воскрешение каждые 5 минут, достаточно в раннем гейме, недорого
  - Куклы Евы (покупаются у Евы в NosVille) : автоматическое немедленное воскрешение, но слишком дорогие для новичка без золота
