# Mejor build de estadísticas para DH

**Q:** Para un DH (Dark Healer), ¿qué build de estadísticas es mejor: 50 de ataque combinado con 90 de elemento, o 34 de ataque combinado con 100 de elemento?

**A:** Para un DH se recomienda priorizar todo en ataque. Como referencia general, hasta el contenido C60 tener 90 en la estadística de elemento escala mejor que otras combinaciones.
