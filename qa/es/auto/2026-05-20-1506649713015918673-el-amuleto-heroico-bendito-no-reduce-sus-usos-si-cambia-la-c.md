# El amuleto heroico bendito no reduce sus usos si cambia la calidad

**Q:** Al usar el amuleto heroico bendito no se quitan las 3 opciones que tiene tras usarlo dos veces, ¿es un bug?

**A:** No es un bug: si al usar el amuleto cambia la calidad del objeto, los usos del amuleto no se reducen. Por eso puede parecer que sigue teniendo las mismas opciones disponibles tras varios usos.
