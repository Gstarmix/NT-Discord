# Diferencia entre el stack de daño con varios mobs y el buff de experiencia creciente

**Q:** Cuando hay muchos mobs, el stack de daño que se consigue, ¿funciona igual que el buff de experiencia creciente? Se cree que es un buff personal (no afecta al enemigo) que se pierde en cuanto termina el buff.

**A:** El stack que se obtiene al pelear contra muchos mobs es efectivamente un buff personal, no algo que se aplique al enemigo, y se pierde al terminar dicho buff. La diferencia respecto al buff de experiencia creciente es que este último tiene una duración fija de 2 minutos.
