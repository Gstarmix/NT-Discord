# Mejor servidor para empezar como jugador nuevo

**Q:** ¿En qué servidor conviene empezar siendo un jugador nuevo?

**A:** Es posible quedarse en NosFire, pero para transferir el personaje a otro servidor se necesitan completar misiones del pase de batalla que resultan complicadas para alguien nuevo. Por eso se recomienda que un jugador nuevo empiece directamente en Dragonveil o Undercity desde el nivel 1, para entender mejor el juego.
