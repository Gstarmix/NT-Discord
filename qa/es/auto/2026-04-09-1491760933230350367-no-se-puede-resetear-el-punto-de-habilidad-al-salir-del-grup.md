# No se puede resetear el punto de habilidad al salir del grupo

**Q:** ¿Qué se puede hacer cuando no se puede resetear el punto de habilidad (SP) tras salir del grupo?

**A:** Como solución alternativa, cierra el juego, cambia el idioma del teclado (por ejemplo a inglés) y vuelve a abrirlo; esto suele permitir que el punto se resetee correctamente.
