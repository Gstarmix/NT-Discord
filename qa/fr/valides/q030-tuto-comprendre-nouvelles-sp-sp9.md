# Q30 : Tuto pour comprendre les nouvelles SP (SP9/10/11) avec leurs jauges ?

**Question FR :** Est ce qu'il y a un genre de tuto pour expliquer toutes ces nouvelle sp avec des points, j'y comprend rien ? (question reformulée : nouvelles SP = SP9/10/11, 3ème génération)

**Réponse FR :**
Les sp9 restent proches des sp classiques, par contre les sp10 et sp11 ont chacune leur propre jauge/ressource thématique qui se remplit quand tu tapes et que tu dépenses sur les gros skills
Par exemple la sp10a joue avec la chaleur, la sp11a avec le gaz, la sp11m avec le carburant, la sp11am avec le courant électrique, la sp11e avec des minéraux
Chaque classe a sa propre logique donc y a pas une règle universelle, faut apprendre la jauge de la sp que tu joues

Pour les détails skill par skill et builds conseillés, regarde ici y'a plein d'explications : https://www.nostar.fr/sp#list

**Contexte :** NosFire 3 (01/04/2026 -> 10/06/2026) + serveurs classiques. Question posée par un joueur qui confond les "points SP" (stats) avec les ressources/jauges des SP 3ème génération.
**Date de validation :** 17/04/2026 (note mise à jour 19/04/2026, précision ressource SP11AM)

**Note :** Ressources identifiées : SP10A = chaleur, SP10AM = carburant (confirmé via screens Hydraupoing, skills Ravitaillement/Coup draconique/Protocole de convalescence), SP11A = gaz, SP11M = carburant, SP11AM = jetons (thème batterie/électrique, cf. skill "Pile d'amplification" qui active les effets de jetons, voir Q42), SP11E = minéraux, SP10E = énergie draconique, SP10M = énergie gravitationnelle.
