# Set apôtre ou gardien, lequel choisir ?

**Q:** Entre le set apôtre et le set gardien, lequel faut-il choisir ?

**A:** Le set gardien complet seul est le plus rentable en rapport investissement/gain. Pour une optimisation poussée si on a beaucoup d'or, la meilleure combinaison est 2 pièces du set apôtre associées au chapeau du set gardien.
