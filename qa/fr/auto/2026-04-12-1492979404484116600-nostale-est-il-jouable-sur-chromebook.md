# NosTale est-il jouable sur Chromebook ?

**Q:** Peut-on jouer à NosTale sur un Chromebook (Chrome OS), sachant que Steam n'est pas disponible sur cette version de Chrome OS ?

**A:** Non, NosTale est un jeu Windows uniquement, il n'existe pas de client natif pour Chrome OS.
