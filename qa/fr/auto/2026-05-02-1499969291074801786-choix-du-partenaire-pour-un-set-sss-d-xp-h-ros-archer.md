# Choix du partenaire pour un set SSS d'xp héros archer

**Q:** Pour un archer h77 visant un rush h80 puis h90 en xp héros avec un set dague, sur quel type de partenaire faut il équiper un set SSS abordable ? Un partenaire de niveau 80 comme Bob refuse l'équipement pour niveau insuffisant.

**A:** Il ne suffit pas de choisir un partenaire de type distance en général : le choix du set SSS dépend surtout de la classe du partenaire, pas uniquement de son type d'attaque. Il faut donc vérifier la classe précise du partenaire avant d'y mettre un set donné.
