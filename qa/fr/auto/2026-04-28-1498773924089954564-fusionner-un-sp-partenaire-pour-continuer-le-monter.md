# Fusionner un SP partenaire pour continuer à le monter

**Q:** Pour continuer à améliorer un SP partenaire S/S/S +39, faut-il obligatoirement le fusionner avec le même SP en S/S/S, ou peut-on utiliser n'importe quel autre SP partenaire comme nourriture ?

**A:** Il faut fusionner le SP partenaire avec le même SP (même type, S/S/S) au palier +39 pour continuer à le faire monter.
