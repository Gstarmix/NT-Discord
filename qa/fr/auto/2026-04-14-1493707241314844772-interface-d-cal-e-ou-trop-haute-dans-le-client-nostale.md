# Interface décalée ou trop haute dans le client NosTale

**Q:** L'affichage du client NosTale semble décalé, avec le haut de l'interface qui dépasse de l'écran, aussi bien en plein écran qu'en fenêtré. Comment corriger ce problème ?

**A:** Il faut vérifier que la résolution configurée dans les paramètres d'affichage de Windows correspond bien à la résolution native de l'écran : un décalage entre les deux réglages peut provoquer ce type de problème d'affichage dans le client.
