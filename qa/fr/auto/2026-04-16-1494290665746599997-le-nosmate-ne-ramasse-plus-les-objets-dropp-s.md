# Le Nosmate ne ramasse plus les objets droppés

**Q:** Mon Nosmate a arrêté de ramasser automatiquement les objets droppés par les mobs, sans que j'aie changé volontairement une option. Comment réactiver ce comportement ?

**A:** Plusieurs pistes ont été proposées sans certitude sur la cause exacte : vérifier si le groupe est en partage de butin (auquel cas il faut le dé-partager), vérifier si le Nosmate a récemment reçu une perle, ou simplement appuyer plusieurs fois sur la touche D pour relancer le ramassage. Si aucune de ces solutions ne fonctionne, il s'agit probablement d'un bug à signaler dans le salon dédié aux rapports de bug.
