# Quels tatouages (loas) utiliser pour un build full SP11 en critique ?

**Q:** Quel tatouage faut-il pour jouer full SP11 en misant sur le critique, et est-ce que le tatouage Exploiter les faiblesses convient aussi ?

**A:** Comme pour toutes les classes qui misent sur le critique, il faut les tatouages (loas) Regard perçant (loa Aigle) et Position d'attaque (loa Lion). Le tatouage Attaque renforcée peut aussi être utilisé. Exploiter les faiblesses n'est pas vraiment adapté à ce build, il est plutôt orienté PvP.
