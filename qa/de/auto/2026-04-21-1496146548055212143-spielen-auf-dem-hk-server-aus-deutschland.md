# Spielen auf dem HK Server aus Deutschland

**Q:** Kann man aus Deutschland auf dem offiziellen HK Server spielen?

**A:** Ja, das ist möglich. Man besucht die offizielle Website des HK Servers, registriert sich dort und lädt das Spiel herunter. Anschließend erstellt man im eigenen Profil ein separates NosTale Konto und hebt die Websperre auf. Es gibt keine Regionssperre, die dies verhindern würde.
