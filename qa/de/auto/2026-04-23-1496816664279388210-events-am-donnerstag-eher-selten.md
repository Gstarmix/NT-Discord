# Events am Donnerstag eher selten

**Q:** Gibt es donnerstags eigentlich normalerweise ein Event?

**A:** Donnerstags finden eher seltener Events statt. Auf dem Server NosFire gibt es davon unabhängig ein dauerhaft laufendes Tattoo-Event, wovon Spieler auf anderen Servern jedoch nicht profitieren.
