# Seelenwächter mit Tropenflügel vs neues Set für PVE Swordman

**Q:** Ist das Seelenwächter-Set mit Tropenflügel im PVE als Schwertkämpfer (Swordi) besser als das neue komplette Set, oder ist das neue Set wirklich so schlecht?

**A:** Das neue Set kommt laut den Antworten nicht annähernd an die Kombination aus Seelenwächter und Tropenflügel heran. Diese Kombination bietet 20% auf 100% Feen-Element sowie einen Abklingzeit-Reset, während das neue Set im Vergleich nur 5% Resistenz gegen Debuffs gegenüber einer Angriffsstufe bringt.
