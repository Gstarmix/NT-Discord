# Umgang mit täglichen Server-Lags

**Q:** Wird es eine Besserung bei den täglichen Server-Lags geben, zum Beispiel im PVP-Arma, wo man dadurch Kisten nicht treffen kann?

**A:** Es handelt sich um ein Softwareproblem im Code. Damit die Ursache gefunden und behoben werden kann, sollten Spieler die Lags über das Report-System melden. Je mehr und je genauer die Meldungen sind, desto höher ist die Chance, dass das auslösende Problem im Code identifiziert werden kann.
