# Unterschied zwischen Map Copies und Map Instances auf NosFire?

**Q:** Gibt es einen Unterschied zwischen den 'Map Copies' auf dem NosFire Server und den im Update vom 16.03 erwähnten 'Map Instances'? Gilt auf NosFire ebenfalls die 120 Sekunden Wartezeit nach einem Mapwechsel, bevor man wieder XP bekommt?
