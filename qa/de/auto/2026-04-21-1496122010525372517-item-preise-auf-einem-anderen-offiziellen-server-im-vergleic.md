# Item Preise auf einem anderen offiziellen Server im Vergleich

**Q:** Wie unterscheiden sich die Item Preise auf einem anderen offiziellen Server (z. B. HK Server) im Vergleich zu unserem Server?

**A:** Einige Items sind dort preislich fast identisch zu unserem Server, allerdings kostet eine Jennifer umgerechnet etwa 45€. Im Basar werden OVR Items für rund 30 Millionen Gold gehandelt, während bestimmte KHS Items bei etwa 35 Millionen Gold liegen. Diese Angaben stammen aus einem informellen Vergleich einzelner Spieler und sind nicht offiziell bestätigt.
