# Wie oft kann man Effekte auf Ausrüstung hochstufen?

**Q:** Gibt es einen Guide dazu, wie oft man einen Effekt auf einem Ausrüstungsstück hochstufen kann, und ab welcher Stufe man S-Effekte bekommen kann? Wenn man auf C-Stufe sowohl erhöhte Krit-Chance als auch erhöhten Kritschaden hat, kann man dann beide hochstufen oder nur einen davon?

**A:** Jeder Effekt lässt sich maximal 2-mal hochstufen, und pro Seltenheitsklasse (C, B, A, S) ist jeweils nur ein Effekt hochstufbar. Hat man auf C-Stufe sowohl erhöhte Krit-Chance als auch erhöhten Kritschaden, kann nur einer der beiden Effekte hochgestuft werden, maximal bis +2.
