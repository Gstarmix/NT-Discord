# Gibt es einen erlaubten Discord Bot für A4 Raid Ankündigungen auf NosFire?

**Q:** Existiert ein Discord Bot, der auf NosFire Akt-5-Raids (A4) ankündigt und in den eigenen Server eingebunden werden kann?

**A:** Es gibt keinen offiziell erlaubten Bot dafür, da Bots grundsätzlich verboten sind. Die Ankündigungen, die man in manchen Servern sieht, stammen aus einer inoffiziellen Datenerhebung im Hintergrund, nicht von einem offiziell nutzbaren Tool.
