# Wie oft kann derselbe Effekt auf Ausrüstung getroffen werden?

**Q:** Wie oft kann ein bestimmter getroffener Effekt auf einem Item vorkommen?

**A:** Ein Effekt kann maximal 2 mal getroffen werden. Es gibt jeweils 2 mögliche Effekte pro Stufe: C, B, A und S. Pro Effektgruppe kann aber nur eine Eigenschaft ausgebaut werden, z.B. bei Stufe C stehen kritischer Schaden und kritische Chance zur Wahl, man kann aber nur eine der beiden Eigenschaften hochziehen, nicht beide gleichzeitig.
