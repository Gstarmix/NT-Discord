# Warum steigen die Preise auf Nosfire so stark?

**Q:** Warum steigen auf dem temporären Server Nosfire die Preise für Items so extrem an, obwohl das doch eigentlich zu Verlusten führen müsste?

**A:** Auf Nosfire kann man zu Serverbeginn viele Dinge sehr günstig kaufen, zum Beispiel einen Seelenwächter für etwa 50 Millionen Gold. Wartet man einige Wochen, kann derselbe Gegenstand auf mehrere Milliarden Gold steigen. Die starke Preissteigerung entsteht also dadurch, dass früh eingekaufte Items im Verlauf der Serverökonomie enorm an Wert gewinnen, nicht durch Verluste der Spieler.
