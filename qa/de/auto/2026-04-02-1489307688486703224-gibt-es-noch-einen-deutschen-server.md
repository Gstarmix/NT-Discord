# Gibt es noch einen deutschen Server?

**Q:** Spielen deutsche Spieler eher auf einem deutschen Server oder auf den internationalen Servern, und wie ist das beim Nosfire-Event gemeint, wenn es dort eine Sprachauswahl gibt?

**A:** Es gibt keinen eigenen deutschen Server mehr, sondern nur noch die zwei internationalen Server (EU-Undercity und EU-Dragonveil). Auch das Nosfire-Event läuft international, die Sprachauswahl bezieht sich nicht auf einen separaten deutschen Server.
