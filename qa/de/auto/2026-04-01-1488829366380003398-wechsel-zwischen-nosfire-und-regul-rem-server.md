# Wechsel zwischen Nosfire und regulärem Server

**Q:** Wo kann man zwischen Nosfire und dem regulären Server wechseln? Geht das direkt im GF Client?

**A:** Der Wechsel erfolgt direkt in der Serverauswahl im GF Client: Dort wählt man einfach den Server Nosfire aus. Wichtig ist dabei, vorher die richtige Sprache auszuwählen, da der Server sonst nicht in der Auswahl erscheint.
