# Kosten der H90 Ausrüstungs-Umwandlung (Hauptwaffe)

**Q:** Was kostet aktuell die Umwandlung von H88 und H90 Ausrüstung, wenn man alle benötigten Materialien kauft?

**A:** Für die Umwandlung der H90 Hauptwaffe werden benötigt: 3x Mullanwaffen-Produktionsschriftrolle (300k je Stück = 900k), 450x Mullan-Werkzeug (135k je Stück = 60,75kk), 250x Abgeschnittenes Kabel (22k je Stück = 5,5kk), 10x Rüstungsplatte (50kk je Stück = 500kk), 100x Wurmzahn (350k je Stück = 35kk), 100x Kristall mit mysteriöser Kraft (5k je Stück = 5kk) und 10x Kraftvoller Timespace-Splitter (1,6kk je Stück = 16kk). Das ergibt aktuell rund 623,15kk Gold, allerdings nur für die Materialien der Hauptwaffe (ohne die Kosten der umzuwandelnden Ausrüstung selbst). Für ein komplettes Set inklusive H88 fallen entsprechend deutlich höhere Kosten an.
