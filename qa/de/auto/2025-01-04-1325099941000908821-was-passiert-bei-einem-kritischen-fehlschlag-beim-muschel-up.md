# Was passiert bei einem kritischen Fehlschlag beim Muschel-Upgrade?

**Q:** Was passiert, wenn man beim Aufwerten einer Muschel einen kritischen Fehlschlag hat? Wird die Ausrüstung zerstört oder fällt sie unter den Ausgangswert?

**A:** Bei einem kritischen Fehlschlag fällt die Muschel um eine Stufe zurück. Von Stufe 0 auf 1 kann kein kritischer Fehlschlag auftreten, dieser Übergang kann also nur normal fehlschlagen. Ab Stufe 1 auf 2 ist ein kritischer Fehlschlag möglich, wodurch die Muschel dann wieder auf Stufe 0 zurückfällt.
