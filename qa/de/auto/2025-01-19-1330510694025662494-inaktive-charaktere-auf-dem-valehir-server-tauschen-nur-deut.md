# Inaktive Charaktere auf dem Valehir Server tauschen, nur deutscher Server wählbar

**Q:** Ich möchte inaktive Charaktere auf dem Valehir Server tauschen, kann im entsprechenden Menü aber nur den deutschen Server auswählen. Wie löse ich das?

**A:** Stelle die Sprache im Client, in dem alle deine Accounts angezeigt werden, auf Englisch um. Danach sollte sich der Valehir Server auswählen lassen.
