# Recupero soldi in caso di truffa nello scambio kk tra server

**Q:** Se scambio kk tra due server con screenshot come prova e la controparte scompare senza rispettare l'accordo, aprendo un ticket è possibile recuperare i soldi? Come ci si può proteggere meglio in questi casi?
