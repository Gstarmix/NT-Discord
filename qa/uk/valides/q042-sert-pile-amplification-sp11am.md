# Q42 : À quoi sert Pile d'amplification en SP11AM ?

**Question EN:** whats this skill in sp11am for, it doesnt give any tokens at all?

**Answer EN:**
the more tokens you have the stronger your attacks and buffs are, and this skill is to activate token effects

if you have skills that use tokens to hit harder or buff/debuff harder, you have to first cast amplification battery then the desired skill, you consume one token and it activates the bonus

you have to redo it every time you want to use a token from a skill

**Context:** NosFire 3 (01/04/2026 -> 10/06/2026) + classic servers. Aquamarin's question on the SP11AM skill "Amplification Battery". Provided screenshots show: holy skill (light fairy), 150 MP, cd 2s, duration 0.1s, 100% to apply the buff "Amplification Battery" (positive effect level 10, duration 10s)
**Validation date:** 19/04/2026
**Theme:** SP11AM / tokens / Amplification Battery / resource activation / 3rd gen SP
**Note:** SP11AM uses "tokens" as resource (battery/electric theme, hence "Amplification Battery" name and "A battery is installed to reinforce the gloves effect"). Particular mechanic: unlike other SP11s where the resource is auto-consumed by empower skills, in SP11AM you have to manually activate Amplification Battery before each token skill to consume 1 token and activate the bonus. Answer level: 2 (casual).
