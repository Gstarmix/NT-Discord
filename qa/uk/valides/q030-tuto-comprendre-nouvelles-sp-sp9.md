# Q30 : Tuto pour comprendre les nouvelles SP (SP9/10/11) avec leurs jauges ?

**Question EN:** is there some tutorial to explain all these new sps with points, i dont get it? (rephrased: new sps = SP9/10/11, 3rd generation)

**Answer EN:**
sp9s stay close to classic sps, but sp10s and sp11s each have their own thematic gauge/resource that fills up when you hit and gets spent on big skills
for example sp10a uses heat, sp11a uses gas, sp11m uses fuel, sp11am uses electric current, sp11e uses minerals
each class has its own logic so theres no universal rule, you have to learn the gauge of the sp you play

for skill-by-skill details and recommended builds, check here theres lots of explanations: https://www.nostar.fr/sp#list

**Context:** NosFire 3 (01/04/2026 -> 10/06/2026) + classic servers. Question asked by a player who confuses "SP points" (stats) with the resources/gauges of 3rd generation SPs.
**Validation date:** 17/04/2026 (note updated 19/04/2026, SP11AM resource clarification)
**Theme:** 3rd gen SP / SP9 / SP10 / SP11 / resource gauges / beginner tutorial
**Note:** Identified resources: SP10A = heat, SP10AM = fuel (confirmed via Hydraulic Fist screens, Refueling/Draconic Strike/Recovery Protocol skills), SP11A = gas, SP11M = fuel, SP11AM = tokens (battery/electric theme, cf. "Amplification Battery" skill which activates token effects , see Q42), SP11E = minerals, SP10E = draconic energy, SP10M = gravitational energy. Answer level: 2 (casual). SP1AM (not SP10AM) uses the haetae/dragon energy switch. Official FR name SP10AM = "Hydraupoing" (not "Poing hydraulique").
