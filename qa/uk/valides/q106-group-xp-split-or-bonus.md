# Q106 : Is group XP divided by the number of members, or is there a group bonus?

**Question :** the exp in group is divided by the X persons in group? or there is any bonus to exp in group?

**Réponse UK :**

both actually, depends on the map

on normal maps it's split, each player gets less per kill
solo = 100%, duo = 75% each, trio = 70% each (as long as everyone deals at least 11% damage)

on lol it's the opposite, grouping gives a bonus to everyone
duo = 110% each, trio = 115% each

lod is even better
duo = 115% each, trio = 130% each

so full trio on lol/lod, but on regular maps you're better off solo if you just want to level one char fast
full breakdown on https://www.nostar.fr/en/xp/guide if you want the details

## Note technique

Source : le guide **[/xp/guide](/xp/guide)** de nostar.fr, valeurs revérifiées le 10/07/2026. Patch officiel Gameforge du 27/06/2023.

**Carte normale :**
- Tous >= 11% dégâts : Duo = 75% each / Trio = 70% each
- Un membre < 11% : Duo = 50% each / Trio = 33% each

**LoL (Terre des Vivants) :**
- Solo : 100% | Duo : 110% (+10%) | Trio : 115% (+15%)
- 0-1% dégâts = 0% XP ; 2-10% dégâts = 10% XP ; ≥11% = bonus complet
- Bonus supprimé dès apparition du boss (MooMoo)

**LoD (Terre des Morts) :**
- Solo : 100% | Duo : 115% (+15%) | Trio : 130% (+30%)
- Si un membre < 11% dégâts : groupe traité comme taille inférieure
- Apparition du boss : tous les gains ×3
