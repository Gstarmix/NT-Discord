# Q69 : Fairy restoration set: how many times can you use it?

**Question (KYO, 23:07) :** Someone has knowlege on fairy restoration set, i know i used them a long time ago and it was limited per fairy, but someone told me u can use as much as u want like solvent, so im wondering if i can spam try or no.

**Réponse UK (Gstar) :**

yeah sp12enjoyer's got it right
fairy restoration set only erases the last effect you got, so it only rewinds one step, and you can buy/use as many as you want, you just can't chain two in a row since there's nothing to erase until you upgrade again and get a new effect

so the loop is exactly that: upgrade, check the effect, if it's bad restore it and try again, if it's good keep going
there's also a hammer that resets the whole fairy back to +0 if you ever want to start over, that's pricier than a restoration set though

nostar.fr has a whole fairy guide + upgrade simulator for this, check https://www.nostar.fr/en/fairies/guide and https://www.nostar.fr/en/fairies/upgrade
