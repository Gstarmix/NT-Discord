# Q56 : Do you get more XP in a party of 3 vs 1 or 2?

**Question :** With a party of 3 u get more exo than 1 or 2? Exp i meant

**Réponse UK :**

depends where you're farming actually

on normal maps the xp is split, each player gets less per kill
solo = 100%, duo = 75% each, trio = 70% each (as long as everyone's dealing at least 11% damage)

on lol it's the opposite, grouping actually boosts xp for each player
duo = 110%, trio = 115%, so a bit more than solo

on lod it's even better
duo = 115%, trio = 130% each

so on lol/lod you want a full trio, on regular maps you're better off solo if you're just trying to level one char fast
full breakdown on https://www.nostar.fr/en/xp/guide if you want all the details

also got all xp bonuses listed at https://www.nostar.fr/en/xp/list if you want to dig into that
there might be a value or two i still need to double check, feel free to flag anything off using the report button at the bottom of the page

## Note technique

Source : le guide **[/xp/guide](/xp/guide)** de nostar.fr
Patch officiel Gameforge du 27/06/2023.

**Carte normale :**
- Tous >= 11% dégâts : Duo = 75% each / Trio = 70% each
- Un membre < 11% : Duo = 50% each / Trio = 33% each

**LoL (Terre des Vivants) :**
- Solo : 100% | Duo : 110% (+10%) | Trio : 115% (+15%)
- 0-1% dégâts = 0% XP ; 2-10% dégâts = 10% XP ; ≥11% = bonus complet
- Bonus supprimé dès apparition du boss (MooMoo)

**LoD (Terre des Morts) :**
- Solo : 100% | Duo : 115% (+15%) | Trio : 130% (+30%)
- Si un membre < 11% dégâts : groupe traité comme taille inférieure
- Apparition du boss : tous les gains ×3
