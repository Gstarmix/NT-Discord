# Q1 : Quelle classe pour un débutant ?

**Question EN:** which class is good for new player?

**Answer EN:**
mage is probably the easiest to start with on nosfire, holy mage (the light sp you get for free) has aoe from the start so you can clear packs without needing to do the sp3/sp4 quests first

swordsman is also solid, berserker carries you all the way to c90 and warrior is free + straightforward

archer is fine but you'll want wild keeper first then demon hunter for aoe, takes a bit more to get going than the other two
