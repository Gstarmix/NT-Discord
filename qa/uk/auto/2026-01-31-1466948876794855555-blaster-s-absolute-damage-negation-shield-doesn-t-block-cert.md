# Blaster's absolute damage-negation shield doesn't block certain area attacks like Arma's laser beam

**Q:** Why does the Blaster skill Drakonium-Mirror Shield (which is described as making the user take 'no HP usage'/absolute damage negation) not prevent damage from Complete Arma's (and likely regular Arma's) 4-directional laser beam? Death can still occur while the shield is active, even though its description implies it should ignore any damage.

**A:** This is most likely not a bug but a property of how certain attack patterns interact with damage-negation effects: since around Act 8+ content, some boss attack patterns are designed to ignore 'no HP/MP consumption' effects, similar to how Asgobas' landing ignores Seer's Eye while Seer can still ignore Draco's landing. Only true absorb effects (e.g. Blade/Shadow, the Water-element passive) and likely counter skills (e.g. MA4) can fully ignore any incoming damage regardless of these newer attack patterns.
