# Closing NosTale when it doesn't appear in Task Manager

**Q:** How can I close the NosTale client if it doesn't show up as a process in Task Manager?

**A:** If the NosTale game process isn't visible in Task Manager, close the GameForge Client instead — closing it will also properly terminate the game.
