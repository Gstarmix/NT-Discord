# How event minibosses spawn on maps

**Q:** How many event minibosses can appear on a single map, what triggers their spawn, and do you need to kill regular mobs to make them appear? Is it a special miniboss that drops event seeds, or a normal one?

**A:** Only one event miniboss can be present per map. Every regular mob has a certain percentage chance to spawn the miniboss instead of respawning normally, so killing mobs is what creates spawn opportunities, but the trigger itself is a chance roll on mob spawn, not a guaranteed result of kills. If a map already has its own regular miniboss (like Kertos Shadow), that miniboss and the event miniboss compete for the same spawn chance, meaning only one of them can appear at a time. This does not apply to act6, since act6 minibosses (like Erenia) are time based rather than mob dependent, so they can spawn independently at any time.
