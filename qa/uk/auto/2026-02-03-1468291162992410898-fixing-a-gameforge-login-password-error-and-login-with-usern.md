# Fixing a Gameforge login password error and login with username only

**Q:** After entering the wrong password too many times on the Gameforge client, it keeps saying the password is wrong even when entering the correct login details. How can this be fixed, and can login be done using just a username instead of an email address (in case the email is forgotten)?

**A:** Repeated failed login attempts trigger a temporary security block, so you need to wait 24 hours before trying again. There is no way to log in using only a username; an email address is required for login, so if the email is forgotten, login is not possible that way.
