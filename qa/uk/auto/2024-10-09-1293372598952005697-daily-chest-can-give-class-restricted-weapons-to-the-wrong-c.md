# Daily chest can give class-restricted weapons to the wrong class

**Q:** I received a limited archer bow from the daily chest even though I'm not an archer. Is this intended?

**A:** This is confirmed, longstanding behavior rather than a new bug: the daily chest can drop class-restricted limited weapons regardless of which class you're playing, so a mage or other non-archer can end up with an archer bow, for example.
