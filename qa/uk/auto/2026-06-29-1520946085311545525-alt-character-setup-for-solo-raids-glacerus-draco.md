# Alt character setup for solo raids (Glacerus/Draco)

**Q:** For solo raids like Glacerus and Draco, do I need a main character plus 14 level 80 alts?

**A:** Yes, that setup is correct: a raid session requires 15 characters total (1 main + 14 alts). For higher efficiency, the recommended long-term setup is 2 main characters plus 28 alts, allowing you to run double raid sessions. Fourteen of those alts should be Martial Artists, since that class provides free extra characters on top of the raid income. Running two sessions this way yields roughly 26 million gold, factoring in gold piles, violet gems, and gillion stones from 5 raids of about 5 minutes each. This is a good way to start, but it is not the most gold-efficient setup long term once you get the hang of it.
