# No class change available for existing characters

**Q:** Can a level 83 mage change class to archer, for example through Mimi? Is there any other option?

**A:** There is no way to change the class of an existing character in this game. Mimi only lets an Adventurer choose to become a Mage, Archer, or Swordsman for the first time; she cannot convert an already-classed character into a different class. The only workaround is to create a new character (the only class that can be created directly at a high starting level is Martial), and keep the original character for other purposes such as raids.
