# Leveling up mid-cast can leave a skill's cooldown stuck

**Q:** When using a skill triggers a job level-up, the skill sometimes gets stuck/bugged, previously fixable by relogging or leveling again. With the new map-hopping cooldown restrictions, relogging is no longer a convenient workaround. Will this be fixed?

**A:** This is a very old, long-standing engine bug present since early NosTale (not specific to any particular server or to the recent map-hopping changes): if the skill you use lands the killing blow that triggers a job level-up, its cooldown can get stuck. The workaround is to use that same skill again before its 'old' cooldown would have expired, which clears the stuck state; relogging is just an alternate workaround. There is no indication this will be fixed.
