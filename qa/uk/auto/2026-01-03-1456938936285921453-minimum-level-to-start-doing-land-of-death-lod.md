# Minimum level to start doing Land of Death (LoD)

**Q:** What level does a low-level character need to be to start joining Land of Death (LoD) now — 55 or 65? Would clearing act 2 be enough preparation?

**A:** The current main questline alone can level a character up to around level 81, so quests can substitute for LoD farming for a long stretch. Beyond that, the 5.2 quest line can push you further, though the mobs encountered there are noticeably tougher.
