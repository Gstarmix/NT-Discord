# Gameforge launcher shows "An error occurred" when starting the game

**Q:** When opening Gameforge Launcher and trying to start playing, an error message ("AN ERROR OCCURRED" popup with Reload/Quit buttons) appears. Is this a known issue and is the team aware of it?

**A:** This is a known issue affecting multiple players as of late April 2026, and the team was made aware and asked to investigate. As a workaround, affected players should also submit a support ticket to Gameforge support (https://support.gameforge.com/) in addition to reporting it in the help channel, since no client-side fix was available at the time.
