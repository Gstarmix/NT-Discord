# Tropic Wings' fairy bonus no longer visibly increases in the character stat window

**Q:** Does the Tropic Wings effect (fairy stat boost) still work? It seems the fairy stat displayed on the character sheet no longer increases when the effect should be active.

**A:** The bonus still applies, but it is calculated on each attack rather than each tick, so it's no longer visible as an increase on the character stat screen. The bonus combines correctly with matching costumes. There is some disagreement on the history: one explanation says older versions showed a visible tick-based increase that was later changed, while another says the on-screen increase was always just a visual display error and the effect always applied only on attack.
