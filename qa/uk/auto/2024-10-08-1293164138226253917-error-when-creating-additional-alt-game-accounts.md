# Error when creating additional alt game accounts

**Q:** I'm trying to create alt accounts (need six total) and get an error telling me to add an existing account manually, but this happens after I've already successfully created two accounts. What's causing it?

**A:** There is a daily limit on how many new game accounts can be created. Once you hit it, the client shows this error; wait 24 hours and you'll be able to create the next account.
