# Amulet of Return cannot bring you from a copy map to the original map

**Q:** Since a recent update, using the Amulet of Return no longer works to return from a copy map to the original map. Is this an intentional change or a bug?

**A:** This is intended, existing behavior rather than a new bug. You can always return to a point you saved while standing on a copy map, regardless of where you currently are. However, you cannot use the amulet to return to a point saved on an original map while you are currently on a copy map. Additionally, if the saved point was recorded on a copy map during an EXP event, the amulet will not teleport you there at all — instead you get randomly teleported within your current map.
