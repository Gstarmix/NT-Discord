# Fastest way to level a new character from around 57 to 65 in Act 2

**Q:** What's the fastest way to level up past main quests for a new swordsman around level 57, before reaching level 65 (67 with the lab)?

**A:** Until level 65 (67 if counting the post-65 lab), doing the main quest line is the most efficient leveling method for a single character, so just push through them. The Land of Death (LoD) instance used to be a much faster alternative, but since early 2025 it has been locked until level 65, so it is unavailable during this level range. Once you reach 65, LoD reopens and can speed things up significantly. There is also a more advanced 'magma multi mule' method for leveling several alts at once, but it requires an already-geared carrying main character and is act 5.2 content, well beyond a new character's needs.
