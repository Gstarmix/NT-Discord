# Timespace ends with 'the portal is closed' message

**Q:** After completing Timespace 3 of Act 8 twice, the timespace does not properly finish and displays 'the portal is closed' instead of letting the player exit.

**A:** This is not a bug. Some timespaces require reading the timespace description before entering, since exiting requires using a Guardian's Magic Orb; without using the orb, the exit portal will show as closed even after the timespace content is completed.
