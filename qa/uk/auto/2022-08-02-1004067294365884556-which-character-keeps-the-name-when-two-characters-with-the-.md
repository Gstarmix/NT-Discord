# Which character keeps the name when two characters with the same name exist across merged servers?

**Q:** I have two characters with the same name on the same account, one on server INT1 and one on INT2. I've logged into both recently, and there's more than a 10-level difference between them (INT2 character is champion level 57, INT1 is champion level 10). When these servers merge and a name conflict occurs, which character keeps the name?

**A:** The character with the higher level (or, alternatively, the one that was created first) keeps the name; the other character will be forced through a rename screen the next time it logs in. This is based on how name conflicts were resolved during a previous French server merge.
