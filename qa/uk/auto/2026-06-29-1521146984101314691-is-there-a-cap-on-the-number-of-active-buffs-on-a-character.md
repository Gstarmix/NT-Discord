# Is there a cap on the number of active buffs on a character

**Q:** Is there a limit to how many buffs a character can have active at the same time, for example when stacking buffs from many different buff alt classes (Crusader, War Knight, Dark Gunner, Holy Angel, Tide Lord, etc.)?

**A:** There is no known hard cap on the number of active buffs a character can have. However, when the buff display was updated to show more than 12 buffs, it was mentioned that the display could bug out past 30 buffs, though this has not been confirmed recently and it is unclear if it was ever fixed. Even if the buff icon display glitches visually past the cap, the buffs themselves remain active.
