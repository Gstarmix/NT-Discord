# NosBazaar search causing client crash (access violation)

**Q:** What causes the client to crash with an 'Access violation... Read of address 00000000' error in NostaleClientX.exe when searching something in NosBazaar?

**A:** This crash is usually caused by a visual/map rendering bug getting triggered in the client. Restarting the client resolves it. This is a known, unfixed client issue as of this report.
