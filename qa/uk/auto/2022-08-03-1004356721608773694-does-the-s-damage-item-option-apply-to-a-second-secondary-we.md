# Does the '+S% damage' item option apply to a second/secondary weapon?

**Q:** Does the '+S% to damage' equipment option also work when using a second weapon (e.g. via dual wielding)?

**A:** It depends on which weapon the option is actually equipped on: if the '+S% damage' option is on the secondary weapon itself, it applies; if it is on the main weapon, it does not carry over to boost the secondary weapon.
