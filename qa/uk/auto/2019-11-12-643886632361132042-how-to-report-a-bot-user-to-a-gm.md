# How to report a bot user to a GM

**Q:** I found a bot user in-game and want to report it to a GM. How do I get support?

**A:** To reach support (e.g. to report a botter), use the command "k!support" in Discord to open a support/report ticket rather than waiting for a GM to be directly available.
