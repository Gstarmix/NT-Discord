# Chef SP cooking smoke effect disappears after using the bazaar

**Q:** After equipping the Chef SP and using it to cook, if I then use the bazaar, the cooking smoke visual no longer appears when I cook. How can this be fixed?

**A:** This is a known issue: after using the bazaar while the Chef SP is equipped, the cooking smoke effect stops showing. To reset it, unequip the SP and walk through a portal. Separately, the Chef SP is also reported to sometimes fail to work properly when there are many players on the same map, so it's recommended to use it in miniland instead.
