# List index out of bounds error when right-clicking speaker/chat item links in HC raid

**Q:** When right-clicking on an item linked via the speaker/chat icon, the game shows a 'List index out of bounds' error or displays a different item than the one clicked. Is this a known bug?

**A:** This is a known issue that occurs after entering a Heroic Challenge (HC) raid. Once inside, right-clicking a previously posted item link in chat can show an empty message, throw a 'List index out of bounds' error, or display an item different from the one actually clicked (offset in the list). In some reported cases it also corrupted chat messages sent before or during the HC raid, making them unusable.
