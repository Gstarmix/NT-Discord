# How Zenas/Erenia time spaces and audiences work

**Q:** How do the Zenas/Erenia time spaces work? An Erenia audience bar filled up from 0 to 100 without triggering a time space — why?

**A:** A time space opens on a random gate about 5 minutes after an audience (Zenas or Erenia) finishes. However, if a time space is already active, or if a second audience is also concluding at the same time, only one of these events can occur at once, so the other gets overwritten/skipped.
