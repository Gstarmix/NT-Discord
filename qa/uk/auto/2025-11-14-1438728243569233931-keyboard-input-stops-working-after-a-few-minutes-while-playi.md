# Keyboard input stops working after a few minutes while playing (Act 5.2 content)

**Q:** While playing 5.2 content (e.g. the 5.2 main quest), keyboard input stops registering after a few seconds to minutes, forcing a game relaunch to fix it, which wastes consumables like oils. Is there a fix?

**A:** This matches the same known input/focus bug reported by other players, linked to the Discord overlay. Disabling the Discord overlay for NosTale is reported to fix keyboard input dropping out.
