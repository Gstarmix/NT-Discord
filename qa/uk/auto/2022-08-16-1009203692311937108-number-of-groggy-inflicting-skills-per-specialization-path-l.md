# Number of groggy-inflicting skills per specialization path (Laby specialist)

**Q:** How many groggy-inflicting skills does the Laby specialist have across her different specialization paths?

**A:** In her base path, Laby has 2 groggy skills: [Oi!] and her CTRL button skill, both usable across all paths. The Ende the World (EtW) path adds [Smash] and [Mystique], for a total of 4 groggy skills. The Raijin/Susano (RaS) path adds [Everybody, Punch!], for a total of 3. The Nightlord (NL) path does not add any extra groggy skill, keeping only the base ones.
