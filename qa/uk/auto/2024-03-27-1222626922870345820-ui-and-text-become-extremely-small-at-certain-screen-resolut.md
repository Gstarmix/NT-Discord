# UI and text become extremely small at certain screen resolutions

**Q:** At certain screen resolutions the game's UI (chat box, HP bar, minimap, text) becomes extremely small and hard to read, making it impractical to use during raids. Is there a fix?

**A:** This is a known display/resolution issue with no clean fix. Turning off full-screen mode brings UI elements closer together but text remains tiny and hard to read. Combining full-screen mode with certain in-game and Windows resolution settings can make the UI bigger, but the result is usually blurry, or it forces changes to the whole system's resolution and window sizes. There is currently no proper solution, only these imperfect workarounds.
