# "The game is currently unavailable to you" error blocking login

**Q:** Trying to log in via the GameForge launcher fails with the message 'The game is currently unavailable to you. Please contact Support', blocking access all day; a support ticket was submitted but got no response for hours. What can be done?

**A:** This message generally indicates a security-related account restriction rather than a client/connectivity bug. There is no in-game fix — the only options are to wait for GameForge Support to process the existing ticket, or submit/follow up on one directly at support.gameforge.com.
