# Why does the character sheet show 0% Shadow resistance despite gear bonuses?

**Q:** My equipment gives elemental resistance bonuses (e.g. a shell effect adding all elemental resistances), but the character stat panel shows 0% for some elements even though it should be higher. Is this a bug?

**A:** This is not a bug: Shadow resistance granted by armor (including shell/costume bonus lines like 'increased all elemental resistances') is not reflected in the main stat panel (P). The underlying resistance is still applied even though the displayed percentage doesn't show it.
