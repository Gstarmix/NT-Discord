# Can you open multiple characters faster than one by one on the Gameforge client?

**Q:** Is there a way to open multiple game characters faster than launching them one by one through the Gameforge client?

**A:** There is no official feature to speed this up. It is only possible using third party programs, and using such tools is against the rules unless the program is explicitly approved by Gameforge. In practice detection would require Gameforge to inspect a player's PC directly, but using unapproved third party software to manage account launching is still considered against the rules.
