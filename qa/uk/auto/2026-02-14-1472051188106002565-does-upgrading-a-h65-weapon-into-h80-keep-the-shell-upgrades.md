# Does upgrading a H65 weapon into H80 keep the shell upgrades and rune?

**Q:** If you upgrade an H65 weapon into an H80 weapon, do you lose the shell upgrades and the rune? If you keep the original weapon, do the shell upgrades stay?

**A:** The rune is kept through the upgrade. The shell upgrades also stay, when crafting you can either pick a random shell or keep the original one, only the rarity changes.
