# NosTale crashes when running another full-screen game (e.g. CS2) at the same time

**Q:** NosTale crashes with an 'Access violation ... in module NostaleClientX.exe' error when another game (e.g. CS2) is running at the same time. Is this a known issue?

**A:** Yes, this is a known issue: NosTale does not handle well having another game running in full screen at the same time, which can cause it to crash with an access violation error. The crash is reported to happen more often when entering a time space while riding a mount.
