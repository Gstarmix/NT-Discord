# libcef.dll loading error on game startup

**Q:** I get a "Error loading libcef.dll, Error code: 0x0000045A" error every time I launch the game. How can I fix it?

**A:** This appears to be a client installation issue. The suggested fix is to fully uninstall the Gameforge client/NosTale, delete the remaining game folder entirely, and then reinstall.
