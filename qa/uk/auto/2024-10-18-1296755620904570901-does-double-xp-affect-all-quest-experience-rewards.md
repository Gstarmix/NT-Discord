# Does Double XP affect all quest experience rewards?

**Q:** During the Double XP event, are quest experience rewards still affected by the multiplier? The event description doesn't seem to reflect this.

**A:** Not all quests have been changed to be excluded from the Double XP multiplier: some quests remain boosted by the event while others are not. The event's description may not fully reflect this inconsistency between quests.
