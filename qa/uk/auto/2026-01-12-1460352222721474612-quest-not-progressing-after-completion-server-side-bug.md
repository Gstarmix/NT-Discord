# Quest not progressing after completion (server-side bug)

**Q:** After finishing a quest step, the follow-up quest didn't appear. Does the whole quest need to be restarted? (It came back after relogging.)

**A:** This is a common bug affecting many or all quests: after completing a task or an entire quest, the follow-up sometimes doesn't trigger. This is likely caused by a server lag that fails to send the update to the client. Simply relogging usually fixes it and the quest progress reappears correctly.
