# Partner personality and technique activation changes revert after restarting the game

**Q:** Changing a partner's personality or activating a partner's techniques works while the game is running, but after closing and reopening the game the change reverts to how it was before. Does this only happen with alts/multiple game instances?

**A:** This happens when playing multiple game clients (alts) at the same time. Settings such as these are saved based on the last NosTale client instance that was closed, so if you run several instances you either need to set the same values on all of them or make sure to close the instance holding the desired settings last.
