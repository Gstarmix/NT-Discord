# Where to find all shell/rune upgrade bonus values

**Q:** Where can all the shell (rune) upgrade bonuses be viewed?

**A:** The full list of rune/shell upgrade bonuses can be checked on the nosapki equipment_runes simulator (Bonuses section at the bottom), or on nostar.fr's shell tables for weapons and armors. Nostar.fr's general equipment simulator also has a dedicated rune/shell upgrade tab.
