# Will a Nosfire character be wiped after the event ends

**Q:** I have an account with a level 85+1 character on Nosfire. Will it be wiped when Nosfire ends?

**A:** If the character is on the Nosfire server and did not reach the required point to transfer it to the main server (BP), it will be wiped when Nosfire ends.
