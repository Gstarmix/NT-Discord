# Error when creating more accounts

**Q:** Is there a current problem with creating accounts? Getting an error when trying to create more accounts.

**A:** This is expected behavior, not a bug: you can create a maximum of 3 accounts per day, after which a 24-hour cooldown applies. If you try to create another account before the cooldown ends, the 24-hour timer resets.
