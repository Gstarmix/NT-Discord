# Swapping fairy element for a no-element skill lowers overall DPS

**Q:** Does it matter which element fairy I use for a skill that has no element (e.g. Renegade's Judgement)? Is it worth swapping fairies mid-raid just to buff such a skill?

**A:** A no-element skill isn't affected by fairy element matching, so swapping doesn't help it directly. More importantly, swapping fairies to boost one specific skill isn't worth it in practice — it doesn't raise overall damage, it lowers it, because your other skills lose their own fairy element bonus. This makes no-element skills like Renegade's Judgement largely dead weight during raids, since players end up avoiding the fairy swap.
