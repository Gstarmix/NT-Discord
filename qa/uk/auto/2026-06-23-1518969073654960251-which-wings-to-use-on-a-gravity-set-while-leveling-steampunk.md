# Which wings to use on a Gravity set while leveling (Steampunk vs Tree)

**Q:** For a Gravity set at +14 with unfinished perfection, should I use Tree Wings (cheaper, only missing the 10% exp bonus) or Steampunk Wings (currently cheaper due to a wheel event) until I add perfection stones?

**A:** One suggestion was to use (retrofitted) Lunar Wings for the time being, and later evolve them into Party or Christmas Wings, rather than committing to either Tree or Steampunk Wings in the meantime.
