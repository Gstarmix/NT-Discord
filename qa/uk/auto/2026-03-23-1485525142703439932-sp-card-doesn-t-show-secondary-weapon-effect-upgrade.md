# SP card doesn't show secondary weapon effect/upgrade

**Q:** When using an SP card with a secondary weapon, the weapon's visual effect isn't displayed, even if the secondary weapon is upgraded to +13. Can this be changed?

**A:** This is not a bug but intended current behavior — secondary weapon effects simply aren't displayed while using an SP card, regardless of its upgrade level. There is already a player suggestion open asking for this to be changed, and players are encouraged to support it.
