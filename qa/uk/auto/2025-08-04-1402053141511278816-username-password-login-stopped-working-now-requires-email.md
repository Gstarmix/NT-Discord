# Username+password login stopped working, now requires email

**Q:** Since a recent client update, logging in with username and password no longer works and now requires the account's email. The previous workaround of launching via the NosTale launcher icon instead of the GameForge client also stopped working. What can be done if the email linked to some of several accounts is unknown?

**A:** This is a known issue as of August 2025, confirmed by multiple players: login via username/password, and the previous workaround of starting the game through the NosTale launcher icon instead of the GameForge client, no longer works — the account's registered email is now required. No client-side fix is known. If you don't know which email is tied to an account, open a support ticket to recover the account information.
