# Land of Legends (LoL) double-reward rest timer

**Q:** Is it a bug that the Land of Legends 15-minute rest timer doesn't allow getting the double reward twice, for example if it was done at 1am and again at 3am the same day?

**A:** This is not a bug, it is normal behavior for now. The double-reward window is tied to the event's official start time rather than to the player's own last participation, so a run completed before the event actually starts will not count toward getting the bonus reward twice.
