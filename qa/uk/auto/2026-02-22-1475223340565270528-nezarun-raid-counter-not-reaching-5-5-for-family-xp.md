# Nezarun raid counter not reaching 5/5 for family XP

**Q:** The Nezarun raid family quest counter stays at 4/5 and never reaches 5/5 for family XP, even after doing multiple raids in the same day.

**A:** This is not a bug: each character's raid completion counts only once per day toward the daily family counter, so repeating the raid multiple times the same day with the same character won't add further progress. To fill the counter, different characters need to each complete the raid once per day.
