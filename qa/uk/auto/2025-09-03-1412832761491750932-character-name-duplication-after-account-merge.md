# Character name duplication after account merge

**Q:** After an account merge, my old archer's name now appears on two level 1 characters on my alt account. Is this expected, and can I get my old name back on my main character?

**A:** This is expected behavior: when merges create duplicate names across characters, only one character can keep the name, and the others are prompted to choose a new name the next time they log in. To secure your old name on your desired (main) character, log into that character first and confirm/keep the name before anyone else takes it. If you already lost the name to another character, you can open a support ticket to ask if it can be restored to you.
