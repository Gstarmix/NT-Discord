# How to change the NosTale launcher/client language without changing server

**Q:** Is it possible to change the language of the NosTale client (e.g. from German to English) without switching to a different game server?

**A:** The language is a launcher setting, not tied to the server: click the planet/globe icon in the launcher's header to change the interface language. Changing the server itself (e.g. from DE to EN) is done separately via the large button next to the account name, and is not required just to change the display language.
