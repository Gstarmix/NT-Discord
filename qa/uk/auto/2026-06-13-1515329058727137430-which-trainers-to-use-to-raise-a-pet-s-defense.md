# Which trainers to use to raise a pet's defense

**Q:** I want to increase the defense of my panda pet. Does the type of trainer used matter?

**A:** Up to +6 or +7 you can use Eva's trainer, but note that this type can both increase and decrease defense. After that point (+5/6/7), switch to pirate or samurai trainers, which will only increase defense. Also remember to set your pet to S (sit/stay) mode so it does not kill the trainers.
