# Rare-level upgrade amulets don't actually give random shells — it's a translation issue

**Q:** The Bronze Hero Amulet and Hero-sized Amulet used for rare-level (rzadki poziom) equipment advancement seem to both give random shell/aura options — is this a bug?

**A:** This is not a bug. The item description itself states that using these amulets during a rare-level upgrade attempt always grants +1 uniqueness level and assigns random shell (aura) options — that is the intended behavior. The confusion came from inaccurate/misleading translations of the item description in the Polish and Italian client languages, not from an actual gameplay bug.
