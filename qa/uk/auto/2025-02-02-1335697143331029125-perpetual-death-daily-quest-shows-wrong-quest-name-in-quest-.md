# Perpetual Death daily quest shows wrong quest name in quest log

**Q:** When accepting the 'Perpetual Death' daily quest, the quest tab displays the name and text of a different quest, 'Endless Cold', instead. Is this a known bug rather than a plain translation issue?

**A:** Yes, this is a known issue: the quest tab incorrectly displays 'Endless Cold' text when the 'Perpetual Death' daily quest is accepted, caused by the quest text being pinned to the wrong entry rather than a simple translation error.
