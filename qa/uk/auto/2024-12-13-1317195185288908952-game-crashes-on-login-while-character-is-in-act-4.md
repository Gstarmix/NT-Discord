# Game crashes on login while character is in Act 4

**Q:** What should I do if my game crashes right after logging in because my character is currently located in Act 4?

**A:** If the client crashes on login specifically because the character is standing in Act 4, submit a support ticket and contact a GM on your server so they can teleport your character back to Nosville. Alternatively, try repairing the game client, as that has resolved this for some players.
