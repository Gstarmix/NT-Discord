# Does shrinking (Tiny Terror) work on the Bear costume set?

**Q:** If I use the Bear costume set as an appearance and apply a shrink/Tiny Terror effect, will it still look small?

**A:** No. Shrink effects only carry over correctly between costumes of the same gender category: male-to-male, female-to-female, and unisex-to-unisex. Since the Bear set is not a unisex costume, it won't shrink properly the way a unisex costume would.
