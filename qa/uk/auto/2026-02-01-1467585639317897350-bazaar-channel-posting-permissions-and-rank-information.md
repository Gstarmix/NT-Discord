# Bazaar channel posting permissions and rank information

**Q:** What do I need to do to be able to post and comment in the Bazaar channel? I used to be able to post but now can't, even at level 7.

**A:** Posting requires Discord level 5, while replying/editing requires level 10, so a level 7 user can post new threads but not yet reply or edit. Searching for 'rank' within the Bazaar channel brings up a thread that explains the level/rank requirements in detail.
