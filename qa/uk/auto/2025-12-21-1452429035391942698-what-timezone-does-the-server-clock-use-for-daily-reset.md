# What timezone does the server clock use for daily reset?

**Q:** What timezone is the official server clock based on? It's hard to figure out when the daily reset happens, and third-party sites like NosApki seem to show the wrong time.

**A:** The official server clock runs on CET (Central European Time), the same timezone as Berlin, which is GMT+1 (or GMT+2 during daylight saving time).
