# 'Connection to the server was lost' immediately on launcher open

**Q:** The game shows 'Connection to the server was lost. NosTale will now close.' as soon as the launcher is opened. What causes this and how can it be fixed?

**A:** This can usually be fixed by repairing the game client and deleting the config.dat file from the game folder.
