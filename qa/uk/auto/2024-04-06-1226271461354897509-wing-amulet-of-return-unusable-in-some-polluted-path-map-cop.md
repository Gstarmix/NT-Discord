# Wing/Amulet of Return unusable in some Polluted Path map copies

**Q:** Why can't I use a Wing of Return (or Amulet of Return) in the Polluted Path map, getting the message "That cannot be used here"?

**A:** This is a known bug: one of the Polluted Path copy maps (present on all channels) is broken and blocks return items from being used. Moving to a different copy of the map resolves the issue.
