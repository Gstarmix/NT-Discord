# Divine Recovery Potion showing as "Limitiert" (Limited) for German client

**Q:** Why does the Divine Recovery Potion appear limited only for players using the German client? Is this an intentional disadvantage for German players?

**A:** This is not an actual restriction, only a translation artifact in the German client text. The item tooltip shows the word "(Limitiert)" as part of the German localization string, but it does not reflect any real 'limited' flag or usage restriction on the item itself.
