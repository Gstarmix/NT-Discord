# Which DH raid monsters cause stun versus blind?

**Q:** In the DH raid, which monsters inflict stun and which inflict blind?

**A:** The large fire type monsters in the DH raid inflict stun (some players find these nearly impossible to avoid, describing it as perma-stun). The Water Knights instead inflict blind, which some players find more annoying to deal with than the stun.
