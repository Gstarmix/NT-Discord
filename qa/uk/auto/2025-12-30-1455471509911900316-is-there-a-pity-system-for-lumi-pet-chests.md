# Is there a pity system for Lumi pet chests?

**Q:** Given the low drop rate for the Lumi pet from chests, is there a pity/guarantee system so players eventually get it after enough openings?

**A:** There is no pity system for the Lumi pet chests. This means it's mathematically possible to open 333 chests (or any number) and still get zero Lumi pets, since each opening is an independent roll with no accumulating guarantee.
