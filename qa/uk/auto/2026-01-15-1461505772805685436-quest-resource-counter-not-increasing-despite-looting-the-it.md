# Quest resource counter not increasing despite looting the item

**Q:** I've been farming for over 2.5 hours and I'm looting the regular resource from mobs, but it never increments my quest's resource counter. Is there a prerequisite or something else I need to do first?

**A:** This happens when two quests' drops conflict with each other — for example, an Act 5.2 quest that requires Valaket egg drops conflicts with an Act 7.2 drop quest. Complete the earlier conflicting quest (A5.2) first, and the intended drops for the later quest should start counting again.
