# Best Necklace/Ring/Bracelet for Swordman c80

**Q:** What are the best necklace, ring, and bracelet for a class level 80 Swordman? Is the 72/75/78 set the correct PvE set?

**A:** There are separate PvP and PvE accessory sets, both usable from class level 72 (ring), 75 (bracelet), and 78 (necklace). The PvE set has enhanced/fairy % bonuses while the PvP set has PvP % bonuses; right-clicking an item shows which type it is. For a PvE build, look for the PvE set in the bazaar within that level range.
