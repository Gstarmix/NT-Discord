# Double FXP event not applying to hardcore raids

**Q:** The double FXP event doesn't seem to have any effect on hardcore raids. Is this a known issue?

**A:** As of this event, several players noticed the double FXP bonus does not seem to apply to hardcore raid rewards, though it does appear to work for other content like Fafnir missions. This was not officially confirmed by GMs, as they are not responsible for event mechanics and Gameforge does not typically respond over weekends, so treat this as an unconfirmed known issue for that event run.
