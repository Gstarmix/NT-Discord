# Missing hero experience buff on an old level 86 account

**Q:** I logged into an old account that is level 86 (not 86+1) and it doesn't have the hero experience buff. How do I get it?

**A:** Unlocking hero levels (CLV) has never been automatic - a character can be max level without ever having hero levels if this step was never done. You need to manually enter Act 6 yourself; doing so is what triggers the game to start granting hero level (CLV) experience.
