# HP/MP regeneration doesn't work outside Miniland while pet trainer SP-training monsters are active

**Q:** If I leave Miniland while my pet trainer's SP-training monsters are still running, my HP/MP regeneration stops. Is this intended behavior?

**A:** Yes, this is intended. As long as the trainer's SP-training monsters are active, the game treats you as if you're in combat, which prevents normal HP/MP regeneration even if you are outside Miniland.
