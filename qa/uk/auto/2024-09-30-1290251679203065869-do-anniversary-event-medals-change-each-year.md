# Do anniversary event medals change each year?

**Q:** I found an anniversary medal item from 2-3 years ago (14th/15th anniversary) — has nothing changed about these medals over the years?

**A:** The anniversary medal keeps the same item ID every year; only its display name is updated to match the current anniversary. It appears to be intended as a permanent item usable in any year, so players can end up holding several medals from different years at once. The more notable part of the anniversary celebration is usually the paid NosMall item (e.g. a mount) plus an accompanying EXP event, though the exact EXP event dates are not always officially confirmed in advance.
