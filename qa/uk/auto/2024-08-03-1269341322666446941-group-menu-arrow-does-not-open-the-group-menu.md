# Group menu arrow does not open the group menu

**Q:** When in a group, clicking the group menu arrow does not make the menu appear. How can this be fixed?

**A:** This happens when another UI element, most likely the buff bar, is overlapping and blocking the group menu button. Go to Settings and use "Reset UI position" to restore the interface layout, which should make the group menu clickable again.
