# SP1 quests no longer give gold except the last one

**Q:** On NosFire, I'm still getting gold and reputation from SP1 quests even though patch notes said gold rewards were removed. Is that normal?

**A:** According to the patch notes, gold rewards were removed from all SP1 quests except the final one in the chain; only the last quest should still grant gold. If gold is still showing up from earlier quests, that contradicts the intended change described in the patch notes.
