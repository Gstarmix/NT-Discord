# Where to find the shell upgrade success rate table?

**Q:** Is there an updated table for the shell upgrade system now that the old official PDF is no longer available?

**A:** A community-maintained table for weapon shell upgrade rates is available on nostar.fr (stuffs/shell page). Official upgrade guide PDFs (including the fairy upgrade one) are hard to find since they aren't distributed through Discord, though this comes up often. Some unofficial collections gather these guides in one place, but sharing those links directly is uncertain territory.
