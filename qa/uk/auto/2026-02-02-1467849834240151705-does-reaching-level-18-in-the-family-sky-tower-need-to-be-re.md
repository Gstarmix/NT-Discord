# Does reaching level 18 in the Family Sky Tower need to be repeated each month to get the reward?

**Q:** If a family already reached level 18 in the Family Sky Tower, will it still get a reward in February, or does the family need to reach level 18 again?

**A:** The reward is distributed per period, so the family needs to reach level 18 again during the current period for the server to grant the reward. If the family does not reach level 18 again and the reward is not distributed, players should contact support, or wait for an official communication if there is a wider distribution issue like in previous months.
