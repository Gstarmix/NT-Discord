# Troubleshooting delayed mob movement / lag not tied to FPS

**Q:** I'm experiencing delayed mob movement (not an FPS drop) even on a good PC and connection, while another PC on the same internet connection doesn't have this issue. Could this be an account-specific problem, and how can it be fixed?

**A:** First check whether the client is capped at 30 FPS instead of 60, as that can look like input/mob delay. Also try setting the NosTale client to run in high performance mode via the graphics card settings (e.g., NVIDIA/AMD control panel), which fixed similar issues for other players. Make sure both NosTale-related processes (e.g., NosTale and NosTaleX) are added to that high-performance list.
