# Rune upgrade event bonus does not apply to shell upgrades

**Q:** An event advertises bonus success chances for engraving runes (Act 7 mechanic) and upgrading runes (Act 4 update), but shell upgrades (+1/+2/+3) still fail at the normal rate. Is the event bugged or does it simply not cover shells?

**A:** The event bonus only applies to normal runes, not to shell upgrades or the 'Shell Master' system. The confusion comes from some translations (French in this case) using the word 'rune' for both the rune system and the shell system, even though they are separate mechanics.
