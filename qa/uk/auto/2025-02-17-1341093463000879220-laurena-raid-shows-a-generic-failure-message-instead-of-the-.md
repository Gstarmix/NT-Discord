# Laurena raid shows a generic failure message instead of the real cause

**Q:** When a Laurena raid attempt fails because all characters got frozen, the game displays a generic 'mission failed' screen instead of a message explaining the actual reason (that the team ran out of usable characters).

**A:** This is a known issue: the game currently shows the same generic failure screen for this case as it does for other raids (e.g. Glacerus) when all characters are frozen, instead of a message specific to that failure cause. There is no fix yet, but it may be addressed for future raids with similar mechanics.
