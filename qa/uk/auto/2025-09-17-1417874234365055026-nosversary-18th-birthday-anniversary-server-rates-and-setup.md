# NosVersary (18th Birthday) anniversary server rates and setup

**Q:** For the NosVersary 18th Birthday anniversary server, is the experience rate double or triple, and how is the server structured?

**A:** The server runs double EXP as the general baseline, with triple EXP active during some weekends. It is essentially a repeat of the Oasis anniversary server format over about 6 weeks, except characters start at level 56 instead of level 1. Expected equipment sets are for level 56 (R5+7) and level 80 (C10 and C25/28). There is no progression book on this server.
