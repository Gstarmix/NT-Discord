# Quest hunting: does the mob or the map location matter?

**Q:** For quests where the target mob location seems unclear, should I focus on hunting the specific unit or going to the listed map location? Does NosAcademy (nosapki) list every subquest?

**A:** What matters for this kind of quest is the mob you need to kill, not the exact map — the quest's description does specify which map to hunt it from, so the quest itself is not bugged despite appearing to mismatch. To be 100% sure of quest details, check NosAcademy (nosapki), though it may not always be complete.
