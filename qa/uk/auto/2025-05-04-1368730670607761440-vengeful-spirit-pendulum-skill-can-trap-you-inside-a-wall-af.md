# Vengeful spirit pendulum skill can trap you inside a wall after a map teleport in raids

**Q:** Is there a known bug where using the Vengeful Spirit Pendulum's third skill at the same time as being teleported to another map or room in a raid (for example the Giant Arma raid) causes you to get stuck?

**A:** Yes, this is a known issue. If you use the pendulum's third skill at the same moment you are teleported to a new map or room, the game tries to return you to your original position, but since that position no longer exists on the new map, you get spawned inside a wall. While stuck, you cannot use skills or mounts to free yourself, and the game reports you cannot use those actions right now. A related and more easily reproducible version of this bug happens when two players enter a portal at the same time.
