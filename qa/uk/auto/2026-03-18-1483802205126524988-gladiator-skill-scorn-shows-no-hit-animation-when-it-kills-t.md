# Gladiator skill "Scorn" shows no hit animation when it kills the target

**Q:** The Gladiator skill "Scorn" (fire-fairy attack buff skill) only shows a hand raising a sword, with no visible hit animation, when it kills the monster. Where does the damage actually come from if there's no effect shown?

**A:** This is known, intended behavior: the skill's hit animation only plays when the target survives the hit; if the skill kills the monster outright, the animation is skipped. The team has decided to leave this as-is for now rather than treat it as a bug.
