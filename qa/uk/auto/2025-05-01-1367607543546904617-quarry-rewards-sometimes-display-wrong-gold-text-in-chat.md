# Quarry rewards sometimes display wrong 'gold' text in chat

**Q:** When collecting rewards from a quarry, the chat sometimes shows 'you received %gold%' instead of the actual material reward. Is this a known bug that should be reported?

**A:** This is a known visual/text display glitch that can happen during periods of high server activity. It only affects the chat message text; no report is needed, and it can be safely ignored as long as the wrong reward was not actually received.
