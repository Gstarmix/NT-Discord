# Why do server updates run for a full day

**Q:** Why does a NosTale update run from early morning until evening?

**A:** Full-day updates are usually caused by data migration for a server merge, the implementation of new event content, and pre-launch testing. Production releases generally include several phases (preparation, migration, release, testing), so they commonly take multiple hours or a full day to complete.
