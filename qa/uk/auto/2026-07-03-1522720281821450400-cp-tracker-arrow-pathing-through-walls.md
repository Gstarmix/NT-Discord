# CP tracker arrow pathing through walls

**Q:** The arrow that guides players to Combat Point (CP) locations sometimes forces them to walk through walls. Has anyone else run into this and is there a fix?

**A:** This is a known issue with the CP-finding arrow. Changing the game's keyboard language setting and restarting the game has been reported to fix the pathing, though the bug can still reoccur on later CPs.
