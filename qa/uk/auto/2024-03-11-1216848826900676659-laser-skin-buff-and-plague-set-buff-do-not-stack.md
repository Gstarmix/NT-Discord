# Laser skin buff and plague set buff do not stack

**Q:** Why doesn't the laser skin buff work together with the plague set buff? When one is active, the other can't be applied at the same time, and vice versa. Is this a bug?

**A:** This is not a bug. The laser skin buff and the plague set buff overwrite each other because they occupy the same buff slot, so only one of the two can be active at any given time.
