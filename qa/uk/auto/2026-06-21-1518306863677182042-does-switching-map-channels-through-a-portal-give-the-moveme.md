# Does switching map channels through a portal give the movement exp debuff?

**Q:** If I move between map zones (for example from Contaminated Zone 1 to Zone 2) using a portal, do I get hit with the exp movement debuff?

**A:** No, moving between maps through portals keeps you on the same map channel, so there is no debuff. This lets you run a double map and return via amulet with no penalty. The exp sickness debuff only triggers when you go through town (which has no channel) and then switch map channel.
