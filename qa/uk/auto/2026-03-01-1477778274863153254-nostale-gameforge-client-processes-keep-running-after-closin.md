# NosTale/GameForge client processes keep running after closing the game

**Q:** After closing NosTale, the game and/or GameForge launcher processes often remain in Task Manager, and the launcher still shows the game as running. Sometimes multiple instances appear. How can this be fixed?

**A:** This is a recurring known issue that tends to appear after certain maintenances and gets resolved by later maintenances. There is no permanent fix; the workaround is to manually end the leftover processes via Task Manager.
