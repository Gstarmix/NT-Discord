# Is Mystic Archer's SP2 meant to be alternated between combat styles?

**Q:** Is Mystic Archer's SP2 skill (with its long cooldown) designed to be alternated between different combat styles, and has this cooldown issue been acknowledged?

**A:** Yes, Mystic Archer SPs are intended to let players switch between combat styles, which is described as the main feature of most Mystic Archer SPs. However, players note that the skill's long cooldown effectively prevents alternating between styles as often as the design seems to intend, which is a known point of frustration.
