# How to unlock raid daily quests (Paimon, Carno, Kiro) after finishing Act 7's first questline?

**Q:** After completing Act 7's first questline, how do you unlock the raid daily quests for bosses like Paimon, Carno, Kiro, and Belial? Is completing the second questline (Celestial Tower) required?

**A:** You should already be able to take the Kiro/Carno dailies without doing the second (Celestial Tower) questline. Check NosAPki for the exact quest prerequisites for each raid daily. The Revenant Paimon daily specifically requires SP9, and there may also be a short secondary quest (such as 'Paimon's Shadow') that needs to be completed first to unlock it.
