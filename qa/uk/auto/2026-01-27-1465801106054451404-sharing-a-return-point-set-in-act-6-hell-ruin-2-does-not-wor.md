# Sharing a return point set in Act 6 (Hell Ruin 2) does not work

**Q:** I tried to share my return point (Hell Ruin 2, Act 6) with lower-level characters, but it didn't work for any of them even though they were all in Act 6. Is this a known issue?

**A:** Yes, this is a long-standing known issue specific to Act 6 — sharing return points there doesn't work, while it works fine in every other act. One theory is that it's caused by changing your return point from a non-CLV area to Act 6 via a shared point, which the game treats as invalid, though the exact cause isn't confirmed.
