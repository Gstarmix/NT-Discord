# Wrong message shown when the Hardcore raid medal expires

**Q:** When the Hardcore (HC) raid medal expires, the game displays a message saying the '17th anniversary medal' has expired instead of the correct medal name. Is this expected?

**A:** This is a known display bug as of early February 2026: the expiration notice incorrectly names the 17th anniversary medal regardless of which medal actually expired. The message will keep showing until the player activates a new HC medal, which is the current workaround.
