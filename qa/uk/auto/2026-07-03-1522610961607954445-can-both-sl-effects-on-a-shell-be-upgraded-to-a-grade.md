# Can both SL effects on a shell be upgraded to A grade?

**Q:** When upgrading shells (runes), is it possible to upgrade both SL effects to A tier, or is only one per tier allowed?

**A:** It is not possible to upgrade both SL effects to A grade at the same time. Only one SL effect per shell can reach the A tier.
