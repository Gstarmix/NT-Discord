# How does character activation/deactivation work after a server merge?

**Q:** After a server merge, can you freely (de)activate characters repeatedly and create new ones, or is this feature limited to only the characters involved in the merge?

**A:** You can activate one of the merge-listed characters and then delete it to free up a character slot for creating a new one. If you are just switching activation status back and forth, the system requires you to swap with another character from the merge list rather than freely toggling.
