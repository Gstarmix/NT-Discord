# Best Sword class SP for Erenia/Zenas raids

**Q:** Which SP (skill build) is best for a Sword (Swordsman) character on Erenia/Zenas raids?

**A:** For Erenia, Crusader/Dark Knight is recommended; for Zenas, Berserker is recommended. In short, Berserker or Crusader work for both raids. More specifically: Berserker excels at debuffs, Crusader at ranged damage, Renegade at melee damage, while Warrior can be good for Ice Cave (IC) content.
