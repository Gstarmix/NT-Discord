# Config window unreachable because it opens outside the screen

**Q:** I cannot open the config menu, why is that?

**A:** This happens when the config window opens outside the visible screen area, an issue also seen in other games and programs. Try adjusting your monitor's display settings to force a window position reset, or search for a general Windows fix for off-screen windows.
