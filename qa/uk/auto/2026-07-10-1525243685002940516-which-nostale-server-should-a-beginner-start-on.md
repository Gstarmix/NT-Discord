# Which NosTale server should a beginner start on

**Q:** What is the best server to start on right now, and is there an up to date beginner guide?

**A:** Opinions differ: Undercity has a larger population and is generally recommended for French/German speaking players, since its community is mostly French and German (non speakers reportedly have a harder time there). Dragonveil may suit players from Italian or Spanish speaking communities better, though it reportedly has more bots and more volatile market prices than Undercity. No community consensus beginner guide was confirmed; a third party progression guide site was mentioned as a resource but wasn't validated by multiple players.
