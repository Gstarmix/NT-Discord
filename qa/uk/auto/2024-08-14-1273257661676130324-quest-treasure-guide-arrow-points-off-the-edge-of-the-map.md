# Quest/treasure guide arrow points off the edge of the map

**Q:** The on-screen guide arrow used to reach a quest/treasure spot is bugged: it points toward the edge of the map instead of the correct location. It was working fine about 10 hours earlier and then broke on every character and every map, with relogging or leaving the map not fixing it.

**A:** This is a known client-side bug affecting the location guide arrow (used for example in treasure hunting). Suggested unverified workarounds include fully restarting the game client (a simple relog is not enough), making sure the PC's system clock is correctly synced, and, if that fails, switching the Windows keyboard input language to a different layout. None of these fixes were confirmed to work by the asker.
