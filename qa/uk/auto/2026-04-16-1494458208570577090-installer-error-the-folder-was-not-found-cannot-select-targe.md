# Installer error "the folder was not found" / cannot select target folder

**Q:** How do I fix the installer error "The action cannot be executed: the folder was not found" (Spanish: "No se puede seleccionar la carpeta como carpeta objetivo") when trying to install/download the game?

**A:** This error commonly occurs when the download/install target folder points to a cloud-synced location (e.g. OneDrive) instead of a normal local folder, which the installer can't use as a target. Repairing or reinstalling the client usually fixes it; in some cases simply rebooting the PC resolves it as well. Make sure the install path isn't set to a cloud-synced folder.
