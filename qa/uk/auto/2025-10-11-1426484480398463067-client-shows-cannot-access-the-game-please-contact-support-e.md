# Client shows "cannot access the game, please contact support" error

**Q:** The game client shows a red banner error saying "You cannot access the game at the moment. Please contact support." How can this be fixed?

**A:** This error sometimes appears without a clear cause. Try closing the client fully via Task Manager (or restarting the whole PC) and launching again, which resolves it for most players. If a restart does not fix it and the message cannot be translated/understood, contact official support directly.
