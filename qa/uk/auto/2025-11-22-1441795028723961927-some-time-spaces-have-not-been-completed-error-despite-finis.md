# "Some time-spaces have not been completed" error despite finishing the required TS

**Q:** While following the NosApki questline on a new character, I completed the first two time-spaces and got the quest for TS57 (Mountain Cave 2), but I get the message "Some time-spaces have not been completed by you," even though I've completed TS56 and part 1 of TS57 twice. Restarting the game and trying with/without a party didn't help. What's wrong?

**A:** This usually means one of the required time-spaces was not actually completed, even if it seems like it was. Check the full list of required time-spaces on nosapki.com/en/ts and verify each one individually to find which one is still missing.
