# PayPal payment error in the NosTale shop

**Q:** When trying to pay via PayPal in the NosTale shop, an error appears, or a 'security measures' loading screen that never finishes. How can this be fixed?

**A:** This PayPal error has reportedly been happening for months with no permanent fix available. As a workaround, try paying through the shop from a web browser instead of the Gameforge client, use the separate 'shop' button available in the client, or try a different account.
