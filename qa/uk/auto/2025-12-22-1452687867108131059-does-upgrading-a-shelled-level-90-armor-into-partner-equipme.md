# Does upgrading a shelled level 90 armor into partner equipment remove the shell automatically?

**Q:** If I have a level 90 armor with a shell on it and I transform it into partner equipment, does the shell get removed automatically, or do I need to use an eraser first?

**A:** The shell should be removed automatically when the armor is transformed into partner equipment, so there is no need to use an eraser beforehand.
