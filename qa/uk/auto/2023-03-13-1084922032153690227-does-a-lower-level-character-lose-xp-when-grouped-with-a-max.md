# Does a lower level character lose XP when grouped with a max-level character via LoL

**Q:** If a level 99+80 character parties with a level 98+68 character using LoL (Level of Loyalty), and both deal a minimum of 11% damage, does the level 98+68 character get full experience, or is there a loss because the level 99+80 character is the max level in the group?

**A:** Yes, there is an experience loss for the lower level character. Since the level 99+80 character counts as level 99, being the max level present in the group, the level 98+68 character loses both normal and reputation (C) experience as a result.
