# Is using GFless (multi-account launcher) allowed on official servers?

**Q:** Is it allowed or bannable to use a tool like GFless, which lets you run multiple Gameforge accounts in one launcher?

**A:** No, it is not allowed. Manipulating the NosTale game client in any way is prohibited, and using or distributing any third-party software not created or authorized by Gameforge is against the rules. This includes bots, hacks, macros, sandboxes, and similar tools, as well as VPNs or proxies that mask your real IP.
