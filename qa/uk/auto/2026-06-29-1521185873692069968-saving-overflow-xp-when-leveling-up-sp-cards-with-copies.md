# Saving overflow XP when leveling up SP cards with copies

**Q:** When upgrading an SP (specialist) card using copies, is there a way to bank or partially use overflow experience, for example using two +100 copies to instantly create another +100, or unlocking a higher level cap gradually while keeping unused overflow XP stored for later instead of losing it?

**A:** There is currently no game mechanic to store or partially use overflow XP when feeding a copy into an SP card, this would only be possible as a suggested feature, not an existing one. As a practical workaround, players recommend leveling a card up to just short of the point where the next copy is required, then feeding in your highest value copy last so its full experience is used efficiently instead of being wasted on overflow.
