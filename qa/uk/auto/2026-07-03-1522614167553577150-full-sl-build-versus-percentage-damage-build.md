# Full SL build versus percentage damage build

**Q:** Is it still better to go for a full SL (specialist level) build rather than a percentage damage build with lower SL?

**A:** It is generally better to combine a decent SL with percentage damage stats rather than maximizing only one of the two. However, while still leveling, having enough SL to enable a good build is sufficient for the time being; the proper percentage-damage gear set can be farmed for later with gold.
