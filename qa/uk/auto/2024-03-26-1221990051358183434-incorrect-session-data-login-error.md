# "Incorrect session data" login error

**Q:** I get an "incorrect session data" error when trying to log in, and it still happens even after changing my password. How do I fix it?

**A:** This is most likely a temporary security block placed on the account. If login still fails after waiting a few hours, the recommended step is to open a support ticket with Gameforge to get the account unblocked.
