# What is the 'progress book' in the NosFire event server

**Q:** In the April 2025 NosFire event server, what is the 'progress book' feature that exists alongside the Battle Pass?

**A:** The progress book is an additional progression track separate from the Battle Pass. Instead of missions, it is built around specific quests, functioning like a personal server-specific achievement/progress tracker.
