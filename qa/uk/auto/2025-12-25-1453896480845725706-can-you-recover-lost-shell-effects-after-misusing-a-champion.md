# Can you recover lost shell effects after misusing a Champion Blessing Amulet?

**Q:** I accidentally used a random Champion Blessing Amulet instead of a static one and lost the shell effects I wanted to keep. Is there any way to get them back, e.g. via a support ticket?

**A:** Yes, this can be recovered by opening a support ticket with the game's customer service. Be aware that the recovery process can take some time to complete.
