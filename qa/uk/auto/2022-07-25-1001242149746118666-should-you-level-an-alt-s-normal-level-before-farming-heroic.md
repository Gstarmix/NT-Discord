# Should you level an alt's normal level before farming heroic/champion level?

**Q:** Is it worth leveling an alt's normal level first before doing heroic level, given the exp penalty from normal level differences in a party?

**A:** There's no need to normal-level an alt separately before doing champion (c-)level content. Champion level is derived from normal level, and champion mobs grant enough normal EXP that grinding c-level already gives the fastest normal-level EXP gain, so you can go straight into c-level farming.
