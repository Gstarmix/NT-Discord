# Bazaar search error when double-clicking too fast

**Q:** Why do I get an error when trying to search for something in the bazaar?

**A:** Clicking on an item listing in the bazaar copies the seller's nickname. If you double-click an item while the bazaar search is still loading, the field is still blank, so the game tries to copy an empty nickname and throws an error. Waiting for the search to finish loading before clicking avoids the issue.
