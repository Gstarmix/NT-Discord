# NosTale fails to open when another game is running

**Q:** NosTale won't open on my PC (issue occurs on two different PCs). How can this be fixed?

**A:** Having another game running in full screen mode commonly conflicts with NosTale and prevents it from opening. If the other game offers a borderless windowed mode, switching to that instead of full screen usually resolves the issue.
