# SSL certificate (not valid) error when launching the Gameforge client

**Q:** Getting an SSL certificate (not valid) error when trying to launch the Gameforge client, even after reinstalling the client and restarting the PC. How can this be fixed?

**A:** This error is usually related to the PC's date and time synchronization, especially when launching through Steam. Open Windows Date and Time settings, make sure no leftover Gameforge client process is still running in the background, enable Set time automatically, and click Sync now, then restart. Some players still see the error only within the Gameforge client itself after a reinstall without it actually blocking play, in which case simply closing the error window is enough to continue.
