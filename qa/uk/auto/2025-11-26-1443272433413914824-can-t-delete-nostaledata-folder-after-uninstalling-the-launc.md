# Can't delete NostaleData folder after uninstalling the launcher

**Q:** After uninstalling the launcher, I still can't delete some folders like NostaleData because it says the folder is open in another program. Why?

**A:** This is usually because a Gameforge client/launcher process is still running in the background even after uninstalling or closing it. Check Task Manager for remaining Gameforge processes and close them before retrying the deletion.
