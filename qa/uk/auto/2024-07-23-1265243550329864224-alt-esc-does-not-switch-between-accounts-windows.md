# Alt+Esc does not switch between accounts/windows

**Q:** Alt+Esc stopped working to switch between NosTale windows/accounts. How can this be fixed?

**A:** This issue is often caused by the game running in true Fullscreen mode. Switching the display setting to Window Mode should allow Alt+Esc (or window switching) to work again.
