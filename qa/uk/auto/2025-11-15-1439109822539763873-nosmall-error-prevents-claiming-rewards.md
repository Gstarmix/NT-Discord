# NosMall error prevents claiming rewards

**Q:** Trying to claim rewards fails with a NosMall error page saying "An error has occurred, close this window and try again," even after sending a support ticket. How can this be fixed?

**A:** This is a known issue as of the message date where claiming rewards through NosMall fails with a generic error page. Suggested workarounds are repairing the game client, switching to a different internet connection, and disabling any VPN, since VPN use can trigger this error. Neither workaround was confirmed as definitively fixing the issue.
