# Can the new hotbar auto-sorting for skills be disabled?

**Q:** Is there a way to disable the newer hotbar behavior that rearranges specialist (SP) skills, so it behaves like before and I don't have to manually drag-and-drop every spell back into place after returning from a long break?

**A:** No, there is currently no option to turn this off. Players who return after time away have to manually reset all their hotbars by hand; no toggle exists for this, though it has been suggested as a possible feature request.
