# How to allocate SP points between element and attack for a mage

**Q:** If an SP has 30 element perfection plus 20 SL element from equipment (with equipment increasing the actual added value), should I allocate 80, 50, or 100 points into element? I'm confused about whether the total should go past 100 for mages.

**A:** What matters is the total shown on the SP card, since SL from element and from gear is already added into it automatically. If gear already contributes 20, allocating 80 points reaches exactly the element breakpoint of 100; element is capped there and only a +20 SP can go beyond it, up to 120, so extra points beyond the cap are wasted and better spent elsewhere. Perfection is a separate stat bonus shown on the card (e.g. a 20/20/20/20 perfection gives roughly +200 attack, +200 defense, +20 element, +2000 HP) and adds on top of the 100 element target, not counted toward it. Priorities differ: for the perfection/gear stat bonus, attack should be prioritized before element for any class including mages, but for raw SP points, a mage should fill element to the 100 breakpoint first, then put remaining points into attack, since attack also grants crit chance/damage and increases the flat attack that fairy elemental damage scales from. Stopping halfway (e.g. at 50) is the worst option since each point is worth more as the count increases. Also, avoid chasing SL before C10 since it is very hard to find on low-level gear.
