# Wing/Amulet of Return unusable in some Polluted Path map copies

**Q:** Why can't I use a Wing of Return or Amulet of Return while in the Polluted Path map?

**A:** This is a known, confirmed bug: only a few of the Polluted Path map copies are affected and block the use of return items. Other copies of the map work fine, so switching copies is a workaround.
