# How to set NosTale to high performance for better FPS

**Q:** How do you set NosTale to high performance mode — is it done through Task Manager's priority setting or something else?

**A:** It is not done through Task Manager priority. Instead, go to Windows Settings, search for "Graphics" (Graphics performance settings), add the NosTale program there, and set it to High Performance.
