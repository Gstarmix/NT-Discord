# Game screen turns into a single solid color / crashes

**Q:** The game repeatedly crashes or shows the screen filled with a single solid color, sometimes causing loss of progress or items (not raid-related). Is there a fix?

**A:** This is a known visual/client issue with no full fix yet, as the root cause is complex to identify even with reproduction steps. When the full-screen single-color bug occurs, pressing Shift+Enter often temporarily fixes it. If the game becomes unresponsive, Ctrl+Alt+Del can help bring up options if Task Manager does not open directly. Repairing game files through the Gameforge launcher's repair tool can also help.
