# Which jewelry set should a C45+ character use?

**Q:** Now that my character is class level 45+, which jewelry/equipment set should I use?

**A:** Get the C15 jewelry set from the Act 7 shop — it gives an attack boost and can be upgraded/changed later at C70 or C90. Swordies and Archers should keep the C15 set until late game, since the C70 set only adds a Fairy boost, which mainly benefits Mages (who should switch to it).
