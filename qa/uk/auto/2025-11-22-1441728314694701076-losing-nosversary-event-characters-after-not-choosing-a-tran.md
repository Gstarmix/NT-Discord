# Losing Nosversary event characters after not choosing a transfer server

**Q:** I didn't log in for a few weeks and lost all my Nosversary event characters. I thought I would be able to choose which server to transfer them to — what happened?

**A:** To be able to select a destination server for Nosversary characters during the merge/transfer, you needed to reach Battle Pass level 15 to obtain the server selection item. Without it, the characters could not be transferred as chosen.
