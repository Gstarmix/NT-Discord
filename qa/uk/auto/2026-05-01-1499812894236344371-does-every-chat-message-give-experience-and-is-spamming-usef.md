# Does every chat message give experience, and is spamming useful

**Q:** Does sending one chat message give one unit of experience? Is spamming messages therefore useless for gaining exp?

**A:** No, spamming messages is useless for gaining experience because there is a cap on how much exp you can earn from chatting per minute. You can check your chat rank and exp progress using the bot command !rank.
