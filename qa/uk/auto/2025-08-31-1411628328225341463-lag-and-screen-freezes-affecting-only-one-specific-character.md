# Lag and screen freezes affecting only one specific character

**Q:** One specific character consistently experiences lag and random screen freezes, no matter which other characters are logged in on the same account. Is this a server-side issue that will be fixed?

**A:** This is generally not a server-side issue but a client/PC-side one. It tends to show up more on characters with more accumulated items/effects/history (like a main character) than on lighter secondary accounts, and can be caused by newer CPU/GPU hardware or Windows 11 not interacting well with the older game client.
