# Garbled/corrupted text (question marks) in shop and item windows

**Q:** Some in-game text (e.g. in the shop/market window) is displaying as garbled characters or question marks instead of proper letters. How can this be fixed?

**A:** This is usually a Windows locale/encoding mismatch rather than a game bug. Check your Windows language and regional settings and make sure the language, display language, and format settings are all consistent, preferably all set to English (US).
