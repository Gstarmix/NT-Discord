# How the 'extra token' text works for raid drops

**Q:** Is the extra-token effect (e.g. from the Alien minipet) bugged? The message says 'Due to the effect, you receive an additional token' but only one token seems to be received.

**A:** This is not a bug. The Hardcore system's item text always includes a quantity number, so when a minipet's extra-token effect triggers you actually see two separate messages: the normal drop message and a second message stating 'You have received an additional token. x1', meaning one extra token was granted. For other raid tokens the base drop text doesn't show a quantity, so only the second 'additional token' message reveals the bonus amount (e.g. receiving 3 of an item would show the normal drop plus an 'x2' additional-token message).
