# Why some skills cannot be learned by certain classes

**Q:** I see a skill that grants magic defense, but the game says it cannot be learned by my character. My friend who plays Archer was able to learn all his skills, so why can't I learn this one?

**A:** That particular skill is restricted to the Swordsman class, which is why it shows as unlearnable for other classes. Each class only has access to its own skill set, so an Archer cannot learn Swordsman skills and vice versa.
