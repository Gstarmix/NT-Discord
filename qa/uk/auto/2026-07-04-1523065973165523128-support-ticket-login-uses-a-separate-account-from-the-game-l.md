# Support ticket login uses a separate account from the game login

**Q:** The support ticket system keeps saying my login data is incorrect even though the same credentials work fine to log into the game via GameForge. How do I fix this so I can open a ticket?

**A:** The support/ticket system does not use the same account as the game login. You need to create a separate support account using your game's email address, then sign in with that new support account to submit tickets.
