# Minimum level to start the Act 6.2 questline

**Q:** What level is required to start the Act 6.2 questline? It could not be started at level 88 after completing all prior acts.

**A:** The Act 6.2 questline currently requires level 95, not 88. A change lowering this requirement was planned but was never fully implemented, so the level 95 requirement is still in effect for now.
