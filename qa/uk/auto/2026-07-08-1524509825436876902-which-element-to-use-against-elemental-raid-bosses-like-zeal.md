# Which element to use against elemental raid bosses like Zealots

**Q:** For content involving Zealot bosses and an elemental rotation, should you always follow a fixed dark to water to light multiplier chain?

**A:** Rather than assuming a fixed elemental rotation, check the specific boss's resistances to decide which element to use, a build such as a c48 character can work well if you can tank the hits reliably. Separately, farming Zealots and Bacooms is easy content even at around character level 10, it's just considered tedious rather than difficult.
