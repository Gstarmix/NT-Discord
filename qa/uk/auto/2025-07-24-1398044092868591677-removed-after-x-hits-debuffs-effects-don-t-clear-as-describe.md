# 'Removed after X hits' debuffs/effects don't clear as described

**Q:** The Merling Curse skill effect is supposed to return the character to normal after 2 hits, but it doesn't. Is this a known issue?

**A:** Known issue as of late July 2025: skills whose description states the effect is 'removed after X hits' (Merling Curse, Heaven Song of Holy, and reportedly others with the same wording) do not clear as described. This appears to affect any skill using that same effect-removal condition. No workaround was given.
