# Which class and SP to invest in as a returning player

**Q:** As a returning player after several years with no idea about the ~30 new SP cards, is there a good class tierlist for raids? Between swordsman and mage, which is better, and which SP should I aim to buy for either?

**A:** Class meta rotates roughly every year (act7 favored swordsman, act8 mage, act9 archer, act10 swordsman again). Mage is currently the easiest and fastest class to deal decent damage since it needs fewer gear pieces to perform well, making it a good pick if you want to be self sufficient. Archer and warrior (melee) become strong once you gather a full set of specific gear, so they require more investment. Ninja is flexible and works with almost any SP if paired with Mecha. For SP choice, it depends on your goals: early on you are mostly questing so there is no need to overinvest in a specific SP yet, and note that an SP rework is expected soon which may change recommendations.
